
# 官场党争复仇游戏 — 需求设计文档

---

## 一、核心定位

- **类型**：回合制策略养成
- **主题**：官场党争 + 复仇主线
- **核心体验**：在朝堂权谋中求生、攀爬、复仇——每一步选择都有代价
- **多结局设计**：根据玩家选择和成就有多个结局（复仇成功/失败/妥协/称霸等）
- **新游戏+**：通关后可继承部分内容重新开始，体验不同路线

## 三、系统一：时间流逝系统

### 基础规则

| 要素 | 设计 |
|------|------|
| **基本单位** | 1回合 = 1月 |
| **时间层级** | 月 → 季(3月) → 年(4季) |
| **回合流程** | 决策阶段 → 结算阶段 → 事件触发 → 下一回合 |
| **行动限制** | 资源点限制——行动消耗资源点，资源点耗尽则无法行动 |
| **跳过回合** | 不可跳过回合，每回合必须完成所有可用操作后自动进入结算阶段 |
| **季节影响** | 弱影响——主要影响事件池和氛围，少量行动条件变化 |
| **时间压力** | 软压力——拖延会产生累积负面后果（仇敌坐大、势力衰减等），但不会直接失败 |
| **全局跨度** | 约30年（360回合） |

### 特殊时间节点

| 节点 | 周期 | 说明 |
|------|------|------|
| **年度述职** | 每年 | 地方官政绩槽评级，决定升迁/罢免 |
| **大朝会** | 稳固值归零时触发（从二品以上） | 从二品以上中枢官员弹劾辩论，决定官员命运 |
| **科举** | 每3年 | 家族成员参加，按宋制三试流程 |

### 大朝会机制

> 当中枢从二品以上官员的官职稳固值被降低到0时，自动触发大朝会，进入弹劾辩论小游戏。

#### 触发条件与参与方

| 要素 | 设计 |
|------|------|
| **触发条件** | 中枢从二品以上官员稳固值降为0 |
| **攻击方** | 造成稳固值归零的党派，该党派中枢所有官员参与 |
| **防守方** | 稳固值归零的官员所在党派，该党派中枢所有官员参与 |
| **攻击方主控** | 造成稳固值最后伤害的官员自动担任主攻方，操控连线实体 |
| **防守方主控** | 稳固值归零的官员自动担任主守方，操控防御技能 |

#### 大朝会结果

| 结果 | 条件 | 效果 |
|------|------|------|
| **防守方被击倒** | 主攻方在时限内将所有逻辑链路连接成功 | 防守方**罢免**，失去官职 |
| **防守方未被击倒** | 时限结束，逻辑链路未全部连接 | 防守方**贬官**（降一阶） |
| **反击成功** | 防守方将主攻方实体血量降为0 | 防守方**保官**，获得3回合稳固值不可攻击buff |

#### 连线小游戏规则

**基本信息：**

| 要素 | 设计 |
|------|------|
| **游戏类型** | 连线小游戏——主攻方操作实体在场地上自由探索，寻找并连接逻辑链路 |
| **总时限** | 60秒 |
| **逻辑链路数量** | 随防守方品级变化：从二品4条、正二品5条、从一品6条、正一品7条 |
| **胜利条件（攻方）** | 在时限内将全部逻辑链路连接 → 击倒防守方 |
| **胜利条件（守方）** | 消灭主攻方实体 → 反击成功；或时限结束未全部连接 → 防守方存活（贬官） |

**主攻方实体：**

| 要素 | 设计 |
|------|------|
| **血量** | 口才 × 3 + 学识（上限300） |
| **移动** | 自由移动，躲避防守方技能 |
| **目标** | 接近逻辑链路节点并连接，每连接1条推进1步，全部连接则击倒防守方 |
| **被消灭** | 血量归零 → 攻击方失败，防守方反击成功保官 |

**防守方技能：**

| 技能类型 | 技能名 | 效果 | 冷却 | 消耗 |
|----------|--------|------|------|------|
| **常规·障碍** | 设障 | 在场地上放置1个障碍物，阻挡主攻方移动路径 | 基础10秒 | 无 |
| **常规·弹幕** | 射弹 | 向主攻方实体发射1枚弹幕，造成5点伤害 | 基础4秒 | 无 |
| **圣眷·伤害** | 雷霆 | 对主攻方实体造成40点伤害 | 无 | 5圣眷 |
| **圣眷·清链** | 断链 | 断开1条已连接的逻辑链路 | 无 | 8圣眷 |
| **圣眷·范围控制** | 天罗 | 全屏放置障碍物 + 3秒内弹幕伤害×2 | 无 | 10圣眷 |

- 常规技能冷却受防守方口才影响：实际冷却 = 基础冷却 × (1 - 口才/300)，最低冷却不低于基础冷却的50%
- 圣眷技能仅消耗防守方家族自身的圣眷，不可由党派其他成员捐献
- 若防守方家族圣眷为0，则无法使用任何圣眷技能，仅能依靠常规技能（障碍/弹幕）防御——圣眷积累是大朝会防御的关键准备

#### 主角参与规则

| 主角身份 | 操作内容 |
|----------|----------|
| **主角为主攻方** | 操作连线实体，自由移动、躲避技能、连接链路 |
| **主角为防守方** | 操作防御技能，释放障碍/弹幕/圣眷技能 |
| **主角不参与** | 不进入小游戏，自动结算 |

#### NPC大朝会自动结算

当主角不参与大朝会时，结果自动计算：

```
击倒概率 = min(0.95, 主攻方口才 / max(1, 主攻方口才 + 守方口才) × (1 + 圣眷差加成))
  - 圣眷差加成 = max(0, (攻方党派总圣眷 - 守方党派总圣眷) / 100 × 0.05)，上限+0.3

反击概率 = min(0.5, 守方口才 / max(1, 主攻方口才 + 守方口才) × (守方圣眷 / (守方圣眷 + 10)) × 0.5)

未击倒概率 = max(0.05, 1 - 击倒概率 - 反击概率)
```

- 击倒概率上限0.95，反击概率上限0.5，未击倒概率下限0.05，确保概率值始终合法
- max(1, ...) 防止双方口才均为0时除零

## 四、系统二：资源系统

### 核心资源模型

| 资源 | 获取 | 消耗/机制 |
|------|------|-----------|
| **资源点** | 每月固定刷新（基础3点）；声望每+1000增加1点，上限10点/月 | 投入建筑产出金钱/声望；行动消耗资源点 |
| **金钱** | 建筑产出；俸禄（按月发放）；事件/任务奖励；私库分红 | 维护声望（抵消衰减）；联姻；结交；其他行动消耗 |
| **声望** | 建筑产出；事件/任务奖励 | 每月衰减（固定+阶梯）；达到阈值提升资源点刷新量；需消耗金钱维护 |
| **圣眷** 🔒 | 步入中枢后解锁；资源点投入建筑产出；官职生产；任务/事件；与皇族关系 | 每月百分比衰减；声望和官职越高衰减越快 |
| **戒心** 🔒 | 步入中枢后解锁；圣眷衰减时转化而来 | 获取圣眷可降低戒心；戒心过高→家族debuff + 概率触发贬官/罢免事件 |

> 🔒 = 步入中枢（正三品以上）后解锁

### 核心循环

```
资源点(每月刷新: 基础3点, 声望每+1000增加1点, 上限10点)
    ↓ 投入建筑
建筑产出 = (建筑基础产出 + 角色能力 × 建筑等级系数) × 资源点加成系数
    ↓ 产出
金钱 / 声望 / 圣眷

声望 → 提升资源点刷新量(每+1000声望→+1点/月) → 更多资源点可用
  ↓
声望衰减(固定+阶梯) ← 需金钱维护(1贯抵1点) ← 金钱

[中枢后]
资源点投入建筑 → 圣眷(百分比衰减) + 戒心(圣眷衰减转化)
圣眷衰减 → ×0.5 → 戒心增长
获取圣眷 → ×0.3 → 降低戒心
高声望/高官 → 加剧圣眷衰减与戒心增长 → 需要平衡经营
戒心过高 → 家族debuff + 概率触发贬官/罢免事件
```

### 资源点

#### 刷新规则

| 要素 | 设计 |
|------|------|
| **基础刷新** | 每月3点 |
| **声望加成** | 声望每增加1000，刷新量+1点 |
| **上限** | 10点/月 |
| **刷新时机** | 每月（回合开始时）刷新，未用完的资源点不累积（不跨月） |
| **用途** | 投入建筑产出金钱/声望/圣眷；执行各种行动消耗 |

#### 建筑产出公式

```
产出 = (建筑基础产出 + 角色对应能力 × 建筑等级系数) × 资源点加成系数
```

| 投入资源点数 | 加成系数 |
|-------------|---------|
| 0点 | 100%（不投入，仅靠角色基础产出） |
| 1点 | 150% |
| 2点 | 200% |
| 3点 | 250%（上限） |

- 每个建筑每回合最多投入3点资源点
- 角色对应能力：金钱建筑看商业能力，声望建筑看学识能力，圣眷建筑看口才能力（详见建筑系统设计文档）
- 建筑基础产出和建筑等级系数详见 [argue_game_architecture.md](argue_game_architecture.md)

#### 行动消耗参考

| 行动 | 消耗资源点 | 说明 |
|------|-----------|------|
| 建筑投入 | 1~3点/建筑 | 每月投入，产出金钱/声望/圣眷 |
| 结交 | 待家族系统确定 | 派出角色结交其他家族人物 |
| 联姻 | 待家族系统确定 | 发起联姻消耗 |
| 攻防稳固值 | 待官场系统确定 | 中枢攻防消耗 |
| 其他行动 | 待各系统确定 | — |

### 声望

#### 数值范围

#### 衰减公式

采用**固定+阶梯**模型：

```
每月衰减量 = 3 + floor(当前声望 / 100) × 3
```

- 基础每月衰减3点
- 当前声望每增加100点，额外衰减3点

| 当前声望 | 每月衰减 |
|----------|-----------|
| 0~99 | 3点 |
| 100~199 | 6点 |
| 500~599 | 18点 |
| 1000~1099 | 33点 |
| 5000~5099 | 153点 |
| 10000~10099 | 303点 |

#### 金钱维护

- 每消耗1贯金钱可抵消1点声望衰减
- 可全额抵消，无上限
- 维护时机：每月结算阶段，先计算衰减量，再根据投入金钱抵消

| 2000~2999 | +2点（共5点） |
| 3000~3999 | +3点（共6点） |
| 4000~4999 | +4点（共7点） |
| 5000~5999 | +5点（共8点） |
| 6000~6999 | +6点（共9点） |
| 7000+ | +7点（共10点，达到上限） |

- 声望每增加1000，资源点刷新量+1

### 圣眷 🔒

> 步入中枢（正三品以上）后解锁

#### 数值范围

- 无上限

#### 获取

| 来源 | 说明 |
|------|------|
| **建筑产出** | 投入资源点到圣眷类建筑，按建筑产出公式计算 |
| **官职生产** | 中枢官职每月自动产出圣眷，品级越高产出越多 |
| **任务/事件** | 特定任务和事件奖励 |
| **皇族关系** | 与皇族（皇族也是一个家族）关系越高，获取圣眷的渠道越多 |

#### 官职圣眷产出（待官场系统确定具体数值）

- 正三品产出最低，正一品产出最高
- 具体数值待官场系统设计时确定

#### 衰减公式

采用**百分比衰减 + 品阶系数倍率**模型：

```
每月衰减量 = 当前圣眷 × 1% × 品阶衰减系数 × 声望衰减加成
```

**品阶衰减系数：**

| 品阶 | 系数 |
|------|------|
| 正三品 | 1.0 |
| 从二品 | 1.2 |
| 正二品 | 1.4 |
| 从一品 | 1.6 |
| 正一品 | 1.8 |

- 每升一阶（从正三品起），系数+0.2

**声望衰减加成：**

- 声望超过10000时，每增加1000声望，衰减系数+0.1
- 公式：`声望衰减加成 = 1 + max(0, (声望 - 10000) / 1000) × 0.1`

#### 衰减示例

| 品阶 | 声望 | 当前圣眷 | 品阶系数 | 声望加成 | 每回合衰减 |
|------|------|---------|---------|---------|-----------|
| 正三品 | 5000 | 200 | 1.0 | 1.0 | 2.0 |
| 正二品 | 8000 | 500 | 1.4 | 1.0 | 7.0 |
| 正一品 | 15000 | 1000 | 1.8 | 1.5 | 27.0 |

这些示例为每月衰减量

### 戒心 🔒

> 步入中枢（正三品以上）后解锁

#### 数值范围

- 0~100

#### 增长机制

**核心公式：圣眷衰减→戒心转化**

```
戒心增长 = 圣眷本月衰减量 × 0.5
```

- 圣眷每月衰减时，衰减量的50%转化为戒心
- 无自然衰减，戒心不会自行降低

#### 降低机制

**获取圣眷→降低戒心**

```
戒心降低 = 本月获取的圣眷 × 0.3
```

- 获取圣眷可降低戒心，比例为1:0.3
- 戒心最低为0

#### 每回合戒心变化公式

```
戒心变化 = 圣眷衰减量 × 0.5 - 本月获取圣眷 × 0.3
```

- 结果为正则戒心增长，为负则戒心降低
- 圣眷衰减与获取在同一月内先计算衰减转化，再计算获取降低

#### debuff阈值与效果

| 戒心区间 | 等级 | 效果 |
|----------|------|------|
| 0~29 | 安全 | 无debuff |
| 30~59 | 警戒 | 圣眷获取速度-10%，家族所有官职产出-10% |
| 60~79 | 危险 | 圣眷获取速度-30%，家族所有官职产出-20%，家族所有官职稳固值被攻击时受伤系数×1.2 |
| 80~100 | 极危 | 圣眷获取速度-50%，家族所有官职产出-30%，家族所有官职稳固值被攻击时受伤系数×2.0 |

#### 贬官/罢免触发概率

| 戒心 | 触发概率 |
|------|---------|
| 80~89 | 每月5%概率触发贬官/罢免事件 |
| 90~94 | 每月15%概率触发 |
| 95~99 | 每月30%概率触发 |
| 100 | 必定触发贬官/罢免事件 |

#### 贬官/罢免执行规则

- 事件触发时，罢免/贬官对象为**家族内官职最高者**
- 家族内多人同品阶时，优先罢免/贬官资历较低者

**戒心降低规则：**

| 处置结果 | 戒心降低量 |
|----------|-----------|
| 每贬低1阶 | 降低5戒心 |
| 免官（完全罢免） | 降低30戒心 |
| 降低上限 | 单次事件最多降低30戒心 |

- 贬官幅度越大，戒心降低越多，但单次最多降30
- 被贬低阶数从当前品阶计算，最低贬至从九品
- 被免官者削职为民——失去品阶和官职，不再领取俸禄，可重新通过科举入仕

#### 戒心博弈核心

```
圣眷衰减 → ×0.5 → 戒心增长
获取圣眷 → ×0.3 → 戒心降低

高品阶/高声望 → 圣眷衰减加快 → 戒心增长加速
                  ↓
            需要持续获取圣眷来压制戒心
                  ↓
            但获取圣眷的建筑投入 → 挤占其他资源点分配
                  ↓
            核心：在圣眷(权力)与戒心(危险)之间找到平衡点
```

### 关键设计点

- **核心决策**：每回合有限的资源点如何分配（哪个建筑投入？投入几点？金钱/声望/圣眷如何取舍？）
- **声望的马太效应与衰减**：高声望带来更多资源点，但衰减更快、维护成本更高
- **金钱维护声望**：1贯抵1点衰减，全额可抵消，但金钱还有其他用途——是否全花在维护上是关键抉择
- **圣眷与戒心的博弈**：圣眷衰减→戒心增长，获取圣眷→降低戒心，形成"越强越危险"的核心张力
- **中枢平衡术**：高品阶高声望加剧圣眷衰减和戒心增长，需要持续经营才能维持权力

## 五、系统三：家族系统

### 核心模型

| 要素 | 设计 |
|------|------|
| **角色卡** | 每个角色是一张卡，有属性、能力，可被放置到建筑中 |
| **建筑系统** | 家族拥有各类建筑，角色卡放入建筑后产出对应收益；建筑种类后续再定 |
| **族长权限** | 族长可操控所有角色卡的放置；非族长成员只能操控自己的卡；只有族长有权驱逐族人 |
| **主控角色** | 玩家操控的角色，可能不是族长（需争取族长之位） |
| **族内关系图谱** | 清晰的族谱/关系网展示，无数值化关系值（纯图谱展示） |
| **资源归属** | 资源点、金钱、声望等均为家族层级拥有，非角色个人 |

### 家族人数上限

| 家族等级 | 声望条件 | 人数上限 |
|----------|----------|----------|
| **初级家族** | 初始 | 10人 |
| **中级家族** | 声望≥3000 | 20人 |
| **顶级家族** | 声望≥7000 | 30人 |

- 家族等级根据声望等级解锁
- 人数到达上限后：适龄夫妻无法生育、无法嫁入人、不会有人来投奔

### 流浪池机制

- 被族长驱逐的族人进入流浪池，不直接视为死亡
- 流浪池中的人每月随机找一个家族请求收留
- 被拒绝后永远不会再次请求该家族
- 家族可在流浪池中主动查看并招募被驱逐的族人
- 族人在流浪池中呆满4个月未被收留则死亡

### 族长更替：竞争继承

族长死亡/退休时，按以下优先级判定继承者：

**有官职者：**

1. **官位品级最高者**继承族长
2. 品级一致时，比拼**资历**，资历高者胜
3. 资历一致时，比拼**学识+口才+军事**三项能力值总和，高者胜

**族内无人做官：**

- 按**长辈→平辈→晚辈**的顺序寻找：
  1. 该辈中**资历最高者**继承
  2. 无资历者，则该辈中**学识+口才+军事**三项能力值总和最高者继承

- 非族长的主控角色需争取族长之位

### 家族间关系

两个家族之间的关系值 = 基础家族关系 + 双方家族所有成年角色（15岁及以上）与对面家族关系的加权平均值

| 角色身份 | 权重 |
|----------|------|
| **族长** | 高权重 |
| **有官职者** | 权重随官职品级提高 |
| **其余成年族人** | 正常权重（1） |

- 仅计算15岁及以上的成年角色
- 族长和官职越高的人对家族间关系影响越大
- 基础家族关系：初始为0，联姻成功后增加
- 基础家族关系会随时间降低直到0：
  - 对方家族声望高于主控家族时，高出越多下降越快
  - 主控家族声望高于对方家族时，不会下降
  - 双方家族分属不同党派时，基础家族关系也会下降

### NPC角色卡关系衰减

- NPC角色卡与主控家族的关系值会自然衰减，每月向0方向靠拢1点（正值每月-1，负值每月+1）
- 结交可提升关系值，事件也可能改变关系值
- 关系值影响：家族间关系计算、NPC家族在党争中的配合意愿

### 家族行动

#### 联姻

- 当家族内有未婚适龄角色时，可向有未婚异性角色的家族发起联姻
- 联姻需付出金钱和资源点
- 联姻成功率受四个维度影响：金钱、资源点、声望、家族关系（公式待定）

**联姻类型：**

| 类型 | 说明 | 角色卡归属 | 难度 |
|------|------|------------|------|
| **婚娶** | 娶对方家族女性入本族 | 对方角色卡加入本家族 | 较高 |
| **招婿** | 招对方家族男性入本族 | 对方角色卡加入本家族 | 较高 |
| **婚嫁** | 本族女性嫁入对方家族 | 本族角色卡加入对方家族 | 较低 |
| **入赘** | 本族男性入赘对方家族 | 本族角色卡加入对方家族 | 较低 |

- 婚娶和招婿要求四维度（金钱/资源点/声望/家族关系）高于婚嫁和入赘
- 联姻成功后基础家族关系增加
- 联姻失败会降低发起方的声望

#### 结交

- 每回合可花费资源点和金钱对其他家族中的人物进行结交
- 需派出家族内一名角色卡执行结交
- 结交会增加目标人物对本家族的关系值
- 结交效果受以下因素影响：
  - 投入的资源点
  - 投入的金钱
  - 家族声望
  - 派出角色的口才属性

### 核心玩法循环

```
角色卡(属性/能力) → 放入建筑 → 产出收益(资源点/金钱等)
                                    ↓
族长统筹所有卡的放置策略 ← 资源回流到家族
玩家若非族长 → 只能操控自己的卡 → 需争取族长权限

家族行动：联姻(获取角色卡) / 结交(提升关系)
```

## 六、系统四：人物/角色系统

### 角色卡结构

| 类别 | 属性 | 说明 |
|------|------|------|
| **基础属性** | 姓名、年龄、籍贯、性别 | 不可变 |
| **能力值** | 学识、军事、商业、口才、样貌 | 数值范围0-100；可成长；刚出生时初始为0 |
| **潜力值** | 学识潜力、军事潜力、商业潜力、口才潜力、样貌潜力 | 后台隐藏属性，不显示给玩家；出生时确定，不可更改；决定能力成长速率 |
| **特性** | 最多4项（来自特性库） | 初始随机/父母继承/事件获取；部分可移除 |
| **头衔** | 无上限（来自头衔库） | 通过放置建筑或担任官职获得 |
| **关系** | 与主控家族的关系 | 仅NPC家族角色卡拥有；数值-100~+100，初始值0 |
| **资历** | 官场资历 | 后台隐藏属性，不显示给玩家；在任时增长，离任后衰减；决定重新入仕所需资源点 |

### 能力值与潜力值系统

#### 潜力值公式

```
潜力值 = (rand(父能力值, 母能力值) / 100 + 1) / 2
```

- 先在父母该能力值之间随机取一个值，再代入公式计算潜力
- 潜力值范围：0.5~1.0（当随机值为0时最低0.5，随机值为100时最高1.0）
- 潜力值在出生瞬间确定，终身不可更改

**举例：** 父学识80，母学识60 → rand(60,80) 如得72 → 潜力=(72/100+1)/2=0.86

#### 潜力值作用

- **潜力值 = 成长速率**：能力上限仍为100
- 潜力值越高，能力成长越快
- 角色刚出生时所有能力值初始为0

#### 能力成长公式

```
每月成长量 = 建筑物基础速率 × 潜力值 × 年龄段速率 × buff加成
```

#### 年龄段速率

| 年龄段 | 速率 | 说明 |
|--------|------|------|
| **1-10岁** | 200% | 孩童期，成长最快 |
| **10-18岁** | 100% | 少年期，正常成长 |
| **18-25岁** | 50% | 青年期，成长放缓 |
| **25-30岁** | 20% | 壮年期，成长缓慢 |
| **超过30岁** | 0% | 不再成长 |

#### 能力成长方式

| 方式 | 说明 |
|------|------|
| **建筑被动成长** | 角色卡放置在对应建筑中时，根据时间流逝自动成长 |
| **资源点增益** | 投入资源点可获得成长增益（buff加成），加速能力提升 |
| **学识成长小游戏** | 主控家族有角色卡在学识成长类建筑中时，每回合可进行限时填字小游戏，获得学识成长加成（详见下方） |

#### 学识成长：限时填字小游戏

**触发条件：** 主控家族有角色卡放置在学识成长类建筑（私塾）中时，每回合触发一次

**游戏规则：**

| 要素 | 设计 |
|------|------|
| **类型** | 限时填字——给出经典语句的上下文，空出关键字，玩家从候选字中选择填入 |
| **时间限制** | 20秒 |
| **题目来源** | 经史子集名句，与游戏世界观契合 |
| **核心机制** | 20秒内填入越多正确字词，加成越高 |
| **跳过** | 玩家可选择跳过小游戏，跳过则默认获得80%的当月学识成长奖励 |

**难度与收益随学识递减：**

建筑内所有角色卡的学识平均值决定填字难度和单次收益：

| 平均学识 | 填字模式 | 候选字数 | 每成功填入1字加成 | 加成上限 |
|----------|----------|----------|-------------------|----------|
| 0-30 | 单字填充 | 4个 | +50% | 200% |
| 31-60 | 双字填充 | 5个 | +35% | 200% |
| 61-80 | 三字填充 | 6个 | +25% | 200% |
| 81-100 | 四字填充 | 7个 | +15% | 200% |

- 平均学识越高→单字加成越低、难度越高（需要填更多字、干扰项更多），但理论上限仍可达200%
- 实际收益：低学识时容易拿满加成；高学识时需要填更多字且每字收益更低，难以达到200%，符合"越学越难"的规律
- 填错不扣减，只是不加成
- 加成作用于该私塾中所有角色的当月学识成长

**加成计算示例：**

| 场景 | 平均学识 | 成功填入 | 加成计算 | 最终加成 |
|------|----------|----------|----------|----------|
| 新手私塾 | 15 | 填对2字 | 2 × 50% = 100% | 100% |
| 新手私塾 | 15 | 填对4字 | 4 × 50% = 200% | 200%（封顶） |
| 中阶私塾 | 50 | 填对4字 | 4 × 35% = 140% | 140% |
| 中阶私塾 | 50 | 填对6字 | 6 × 35% = 210% | 200%（封顶） |
| 高阶私塾 | 75 | 填对6字 | 6 × 25% = 150% | 150% |
| 顶级私塾 | 90 | 填对8字 | 8 × 15% = 120% | 120% |

### 特性系统

#### 特性等级

| 等级 | 生成概率 | 说明 |
|------|----------|------|
| **寻常** | 40% | 普通特性 |
| **清良** | 30% | 较好特性 |
| **翘楚** | 20% | 稀有特性 |
| **绝伦** | 10% | 顶级特性 |

- 每个等级有对应的特性库

#### 特性生成流程

角色生成时依次判定是否生成特性，最多4项：

```
第1个特性：60%概率生成 → 若生成则继续判定第2个
第2个特性：40%概率生成 → 若生成则继续判定第3个
第3个特性：20%概率生成 → 若生成则继续判定第4个
第4个特性：10%概率生成 → 结束
```

- 判定为"生成"后，再按四个等级的生成概率（寻常40%/清良30%/翘楚20%/绝伦10%）判定等级
- 判定等级后，从对应等级的特性库中随机抽取一个特性赋予角色

#### 特性继承（生育时判定，优先于随机生成）

- 父母双方的所有特性均有50%概率遗传给孩子
- 按特性等级从高到低依次判定（绝伦→翘楚→清良→寻常）
- 满4个特性后不再判定
- 若父母所有特性遗传判定结束后孩子特性未满4个，再进行随机特性生成判定

#### 特性获取与移除

| 要素 | 设计 |
|------|------|
| **获取** | 父母特性继承（优先） / 初始随机生成 / 后期事件获得 |
| **移除** | 部分特性可通过特殊事件/条件移除（非全部） |
| **效果** | 每个特性对能力值产生修正 |

### 传承机制

| 传承内容 | 规则 |
|----------|------|
| **特性** | 父母特性50%概率遗传，按等级从高到低判定 |
| **头衔** | 仅世袭头衔（给予家族的）可传承给嫡长子，其余头衔不传承 |

### 角色卡获取

| 方式 | 说明 |
|------|------|
| **生育** | 家族内繁衍后代，获得新角色卡 |
| **婚姻** | 联姻获取新角色卡（入赘/嫁入） |
| **投奔** | 外来角色主动投奔家族 |

### 婚育年龄

| 性别 | 可婚育年龄 | 说明 |
|------|------------|------|
| **男性** | 15-60岁 | 超出此范围无法生育 |
| **女性** | 16-40岁 | 超出此范围无法生育 |

- 只有双方均在各自婚育年龄段内时才可怀孕生子
- 任何一方超出范围都无法生育

### 生育概率

- 男女结婚的那个月（回合）开始计算
- 每月进行一次怀孕判定
- 判定概率：首次20%，每成功怀孕一次后概率减少5%
- 判定为怀孕后冷却24个月（2年），期间不再进行怀孕判定
- 冷却结束后重新进入判定周期，概率按已怀孕次数递减

### 角色卡移除

| 原因 | 说明 |
|------|------|
| **死亡** | 年龄/事件导致角色死亡 |
| **出嫁** | 联姻时角色出嫁离开家族 |

### 年龄与死亡系统

- 角色寿命范围：0-100岁
- 年龄随游戏时间推进增长
- 传承机制确保老角色退场后新角色接替

#### 死亡概率（每月判定一次）

| 年龄段 | 死亡概率 | 说明 |
|--------|----------|------|
| **0岁（出生时）** | 1% | 仅在出生时判定一次，夭折率 |
| **1-10岁** | 0.15%/月 | 孩童期 |
| **11-39岁** | 0.03%/月 | 青壮年期 |
| **40-49岁** | 0.15%/月 | 中年期 |
| **50岁以上** | 递增，2次方增长 | 100岁时概率为100% |

#### 50岁以上死亡概率公式

```
死亡概率 = ((年龄 - 50) / 50) ^ 2 × 100%  （每月判定）
```

| 年龄 | 月死亡概率 | 年化死亡概率 |
|------|-----------|-------------|
| 50岁 | 0% | 0% |
| 60岁 | 4% | ~39% |
| 70岁 | 16% | ~87% |
| 80岁 | 36% | ~99%+ |
| 90岁 | 64% | ~100% |
| 100岁 | 100% | 100% |

### 官制框架：九品十八阶

| 要素 | 设计 |

| **品阶** | 九品，每品分正从（正X品、从X品），共十八阶 |

### 科举系统

#### 基本规则

- 每3年举行一次
- 家族成员可参加科举，获取做官身份
- 不产生新角色卡，仅赋予现有角色官场资格
- 三试跨年进行：解试在科举年8月，省试在次年2月，殿试在次年3月
- 已有官职者禁止再参加科举
- 同一角色一生最多参加3次科举（主角仅1次机会，不中即游戏结束）

#### 参加条件

| 条件 | 说明 |
|------|------|
| **年龄** | 15岁及以上 |
| **性别** | 无限制，男女同等条件参加 |
| **科举资格** | 未参加过科举者方可参加（散官已经参加过科举，不可再参加） |
| **学识** | 参加科举无学识门槛（解试需学识≥50方可通过） |
| **报名费** | 每试5贯（解试5贯，省试5贯，殿试5贯） |
| **报名时机** | 每试前1个月报名缴费 |
| **次数限制** | 每人一生最多参加3次科举（主角仅1次机会，不中即游戏结束） |

#### 科举时间线

```
科举年7月  → 报名解试（7月报名缴费，8月考试）
次年1月     → 报名省试（1月报名缴费，2月考试，当月出榜）
次年2月     → 报名殿试（省试出榜后当月报名缴费，3月考试）
```

- 省试当月出榜：2月考试结束后当月公布成绩，通过者当月即可报名殿试

#### 第一试：解试（州试）

**规则：**

- 在各州分别举行，按州独立判定
- 仅考察学识属性
- 门槛制：学识≥50即可通过，获解额
- 未通过者下次科举需重新参加解试

**判定方式：**

```
学识 ≥ 50 → 通过，获解额
学识 < 50 → 未通过
```

- 无名额上限，只要学识达标即通过

#### 隐藏分机制

省试与殿试的评分公式均附带一个**隐藏分**，玩家不可见。

**基本规则：**

- 每位考生每次科举时随机生成一个0.1~0.9的隐藏分（均匀随机）
- 隐藏分正常情况下**不参与**评分公式计算
- 仅在综合分相同时，比较隐藏分作为排名依据，隐藏分高者排前
- 隐藏分仍相同则随机决定

**礼部尚书操控：**

- 拥有**礼部尚书**官职的家族可操控科举隐藏分
- 该家族可将指定考生的隐藏分设为大于1的值（具体数值由家族投入资源决定，待官场系统确定）
- 当隐藏分 > 1时，该隐藏分**直接加入评分公式**参与计算

**影响：**

| 隐藏分范围 | 作用 |
|-----------|------|
| 0.1~0.9（正常） | 不参与评分，仅同分时作为排序依据 |
| >1（被操控） | 直接加入评分公式计算，显著提升排名 |

**示例：**

- 省试正常：`综合分 = 学识 × 0.6 + 口才 × 0.4`，同分时比隐藏分
- 省试被操控：`综合分 = 学识 × 0.6 + 口才 × 0.4 + 隐藏分`（隐藏分>1时生效）
- 殿试正常：`综合分 = 学识 × 0.5 + 口才 × 0.3 + 声望排名分`，同分时比隐藏分
- 殿试被操控：`综合分 = 学识 × 0.5 + 口才 × 0.3 + 声望排名分 + 隐藏分`（隐藏分>1时生效）

#### 第二试：省试（礼部试）

**规则：**

- 全省（3路合并）考生统一排名
- 考察学识与口才两项属性
- 取综合分最高的前10名通过，进入殿试
- 未通过省试者保留解额2届，下次科举可免解试直接参加省试

**判定方式：**

```
正常：综合分 = 学识 × 0.6 + 口才 × 0.4
被操控：综合分 = 学识 × 0.6 + 口才 × 0.4 + 隐藏分（隐藏分>1时）
```

- 全路考生按综合分降序排列 → 取前10名通过
- 同分时比较隐藏分，隐藏分高者排前
- 隐藏分仍相同则随机决定

**解额保留规则：**

- 通过解试但未通过省试者，解额保留2届
- 保留期间内下次科举可免解试直接参加省试
- 超过2届未通过省试，解额作废，需重新参加解试

#### 第三试：殿试（皇帝亲试）

**规则：**

- 全省（3路合并）10名省试通过者统一排名
- 考察学识、口才与家族声望三项
- 排名前3为三甲，直接获官职
- 第4~10名获做官身份 + 50声望奖励，需通过空缺填补机制获取官职

**判定方式：**

```
正常：殿试综合分 = 学识 × 0.5 + 口才 × 0.3 + 声望排名分
被操控：殿试综合分 = 学识 × 0.5 + 口才 × 0.3 + 声望排名分 + 隐藏分（隐藏分>1时）
```

**声望排名分计算：**

- 同家族考生声望排名分相同
- 按家族声望在殿试考生中排序

| 家族声望排名 | 排名分 |
|-------------|--------|
| 第1名家族 | 100 |
| 第2名家族 | 80 |
| 第3名家族 | 60 |
| 第4名家族 | 40 |
| 第5名及以后家族 | 20 |

- 10名殿试考生按综合分降序排列 → 第1~3名为三甲
- 同分时比较隐藏分，隐藏分高者排前
- 隐藏分仍相同则比较声望排名分，高分者排前

#### 殿试三甲与官职

| 等级 | 名称 | 人数 | 结果 |
|------|------|------|------|
| **一甲** | 进士及第 | 1人 | 直接获官职（从八品起） |
| **二甲** | 进士出身 | 1人 | 直接获官职（正九品起） |
| **三甲** | 同进士出身 | 1人 | 直接获官职（从九品起） |
| **未入三甲** | — | 7人 | 获做官身份 + 50声望，需通过空缺填补机制获取官职 |

#### 三甲官职分配机制

按优先级依次判定：

1. **有空缺**：对应品级有空缺官职则直接填补
2. **替换丙等者**：对应品级无空缺时，若有去年述职丙等者，直接替换，被替换者变为散官回家
3. **资源点替换**：前两者都不满足时，家族需投入资源点替换在任者，被替换者变为散官回家

#### NPC家族科举行为

- NPC家族中符合条件的成员由AI自动报名参加科举
- 报名费从NPC家族金钱中扣除
- NPC考生的能力值按其角色卡属性参与判定，与玩家同等规则


### 地方官升迁：政绩槽机制

| 要素 | 设计 |
|------|------|
| **政绩槽** | 地方官职专属，与官职绑定而非角色；中枢官职无政绩槽，改为稳固值机制 |
| **填充速率** | 基础速率由政绩槽依据能力决定 |
| **速率加成** | 声望、党派支持、资源投入、圣眷均对速率有加成 |
| **事件影响** | 事件可直接增减政绩槽进度 |
| **年度述职** | 每年回京述职，评价政绩槽填充程度 |
| **升迁条件** | 任期内任意连续两年甲等（槽填充≥100%）方可升迁，不要求必须是任期前两年 |
| **连升机制** | 连续两年甲等时，根据政绩槽超额程度决定连升阶数：两年均≥100%升1阶；两年均≥150%升2阶；两年均≥200%升3阶 |
| **升迁方向** | 连续两年甲等后可选择进入中枢或继续留任地方（品级更高） |
| **中枢后** | 进入中枢官职后无政绩槽，改为稳固值机制 |

### 年度述职评级

| 等级 | 条件 | 效果 |
|------|------|------|
| **甲等** | 政绩槽 ≥ 100% | 升迁必要条件 |
| **乙等** | 80% ≤ 政绩槽 < 100% | 安全，但不计入升迁 |
| **丙等** | 60% ≤ 政绩槽 < 80% | 警告，连续两年丙等→罢免 |
| **丁等** | 政绩槽 < 60% | 直接罢免 |

### 关键规则

- **升迁**：任期内任意连续两年甲等方可升迁（不要求必须是任期前两年）
- **连升**：两年均≥150%可连升2阶，两年均≥200%可连升3阶
- **罢免触发**：连续两年丙等 或 有一年丁等
- **政绩槽重置**：每年述职后清零重新累积

### 地方官任期

- 地方官每3年一任
- 3年内未获晋升者需平级调任至其他同品级官职
- 若同品级所有位置已满或被晋升者占据，则保留品阶成为散官回家休养

### 散官机制

- 散官：有品级但无官职的官员
- 保留品阶和俸禄
- 资历仍会缓慢降低（与致仕/罢免在家相同）
- 可通过空缺填补机制重新获取官职（需根据资历判定投入资源点）

### 女性官员孕期机制

由于此游戏中男女均可科举做官，女性官员在怀孕期间适用以下特殊规则：

**触发条件：** 女性官员怀孕判定成功后，从怀孕第6个月开始生效

**孕期时间线：**

```
怀孕判定成功 → 前6个月：正常担任官职，享受全部收益
              → 第7~10个月（共4个月=4回合）：进入孕期休假状态
              → 第10个月孩子出生：休假结束，恢复正常官员状态
```

**孕期休假状态规则：**

| 要素 | 规则 |
|------|------|
| **角色位置** | 从官职上回到家族中 |
| **官职归属** | 视为该角色依然在官职上，官职不产生空缺 |
| **稳固值/政绩槽** | 依然可以被攻击和防御 |
| **官职收益** | 不享受官职提供的任何收益（俸禄、圣眷产出、家族加成等） |
| **政绩槽填充** | 孕期休假期间政绩槽不增长 |
| **任期计算** | 孕期休假时间计入任期 |
| **资历** | 孕期休假期间资历视同散官，会逐渐降低 |
| **稳固值被攻破** | 若女性官员在中枢官职上，休假期间稳固值被攻击降至0，同样会被贬官或罢免 |

**休假结束：**

- 孩子出生后，角色自动恢复官员状态，回到官职上，重新享受全部收益
- 无需通过空缺填补机制重新获取官职

### 地方官空缺填补

- 官职因死亡/罢免/任期届满产生空缺后，各家族可投入资源点获取该官职
- 未担任官职的角色需完成过科举方可获取空缺（入仕）
- 不同获取方式所需资源点不同：

| 获取方式 | 说明 | 资源点消耗 |
|----------|------|------------|
| **升迁** | 从低品级官职升任高品级空缺 | 中等 |
| **调任** | 同品级官职之间平调 | 较低 |
| **自降获取** | 主动降品级获取空缺官职 | 最低 |

- 多个家族竞争同一空缺时，投入资源点最高者获得
- 罢免/致仕/散官在家后重新入仕，需根据资历判定需投入的资源点

### 资历系统

- 角色隐藏属性，不显示给玩家
- 用于判定重新入仕时需投入的资源点数量

| 状态 | 资历变化 |
|------|----------|
| **担任官职** | 资历增加，官职越高增加越快 |
| **致仕/散官/罢免在家** | 资历缓慢衰减 |

- 资历越高，重新入仕所需投入的资源点越少
- 资历越低，重新入仕所需投入的资源点越多

### 中枢官职：稳固值机制

| 要素 | 设计 |
|------|------|
| **稳固值** | 每个中枢官职拥有（与官职绑定而非角色），范围0-100，初始值50 |
| **提升己方** | 消耗资源提升己方官职的稳固值 |
| **攻击敌方** | 消耗资源攻击非己方官职，降低其稳固值。攻击方式包括普通攻击和弹劾 |
| **罢免/贬官** | 稳固值降至0时：从二品以上触发大朝会辩论；正三品直接贬官（降一阶）或罢免（由戒心等级决定，戒心≥80则罢免，否则贬官） |
| **空缺升迁** | 出现空缺官职时，可消耗圣眷+声望请求升迁调任 |

### 弹劾机制

弹劾是攻击中枢官员稳固值的一种特殊形式，与普通攻击独立，不走大朝会流程。

| 要素 | 设计 |
|------|------|
| **发起条件** | 主角或NPC中枢官员均可发起；需消耗大量资源点+金钱；目标必须为非己方官员 |
| **弹劾效果** | 对目标官员稳固值造成一次性大幅降低（远超普通攻击），具体数值待稳固值攻防系统确定 |
| **弹劾代价** | 发起方消耗大量资源点和金钱；弹劾的成功/失败判定条件待稳固值攻防系统确定 |
| **与普通攻击的区别** | 弹劾是集中式高消耗高伤害的攻击手段；普通攻击是持续式低消耗低伤害的消耗手段 |
| **与大朝会的关系** | 弹劾≠大朝会。弹劾仅降低稳固值，若稳固值降至0则按品级触发后续（大朝会或直接贬官/罢免） |

### 贬官机制

- 大朝会贬官：降一阶（如从二品→正二品），保留官职
- 正三品稳固值归零贬官：降一阶，保留官职
- 戒心触发贬官：按戒心降低规则执行（每贬一阶降5戒心，单次最多降30）
- 罢免：削职为民，失去品阶和官职
- 贬官后可复出，但需要重新积累

### 中枢玩法循环

```
资源投入 → 攻击敌方稳固值 / 提升己方稳固值
                ↓
敌方稳固值=0 → 从二品以上：触发大朝会辩论
             → 正三品：直接贬官/罢免（视戒心等级）
                ↓
贬官/罢免 → 产生空缺
                ↓
消耗圣眷+声望 → 请求升迁填补空缺
```

## 八、系统六：党争系统

### 基础设定

- 2个对立党派（固定，不可完全消灭，只会式微）
- 主控家族加入党派后开启党派系统
- 后期可通过特殊事件/剧情触发脱离，自立门户成立第三党

### 党派领袖

- 一个党派有多位领袖
- 前任领袖死亡/贬官/罢免后，从以下三个维度判定接任：
  - 官职高低
  - 声望
  - 党派贡献

### 党派资源

| 资源 | 来源 | 用途 |
|------|------|------|
| **党派声望** | 中枢官员官职产出（越高官职产出越多）；成员家族上缴声望；地方官员政绩影响；被罢免/贬官则降低 | 解锁党派机构；兑换临时党派资源点 |
| **党派资源点** | 消耗党派声望获得；党派机构产出 | 攻击/防守官职稳固值；补充己方地方官政绩槽；投入党派机构提升产出 |

### 党派机构（党派声望达标后解锁）

| 机构 | 机制 | 产出 |
|------|------|------|
| **谍报机构** | 抽卡制，每回合随机产出情报卡，投入资源点越多产出越多 | 多类型情报卡，配合资源点攻击非己方官员 |
| **舆论机构** | Buff制，投入资源点产生buff | 己方增益buff / 敌方减益buff，投入越多越强 |
| **私库** | 金钱库，己方官员收入按比例进入；可主动存入换党派贡献 | 上交皇帝→党派领袖获圣眷；每年按官位+贡献分红给成员→提高忠诚 |

### 党派行动

| 行动 | 消耗 | 效果 |
|------|------|------|
| 攻击/防守官职 | 家族资源点 | 攻击非己方官职/帮助己方，获得党派贡献 |
| 帮助非主控家族 | 家族资源点 | 获得党派贡献 + 提高该家族党派忠诚度 |
| 党派领袖攻防 | 党派资源点 | 帮助非主控家族防御获党派忠诚度 |
| 拉拢策反 | 家族资源点/金钱 | 降低目标家族与所属党派忠诚度 |

### 党派贡献

| 用途 | 说明 |
|------|------|
| 换取情报 | 在谍报机构获取情报卡辅助攻防 |
| 兑换资源 | 换算为金钱/声望（从私库/党派声望扣除） |
| 年度分红 | 每年从私库按比例获得 |

### 党派忠诚

| 要素 | 设计 |
|------|------|
| **初始值** | NPC家族加入时默认70（0-100） |
| **自然衰减** | 每年自动降低 |
| **提升** | 帮助该家族防御/分红等行为 |
| **降低** | 被拉拢/利益受损等 |
| **策反阈值** | 忠诚度<30时可被拉拢策反 |
| **叛离** | 忠诚度=0自动退出并加入对立党派 |

### 党派覆灭规则

- 固定2个党派不可完全消灭，只会式微
- 主控可成立第三党

## 九、系统七：事件系统

### 事件类型（多类型混合）

| 类型 | 触发方式 | 说明 |
|------|----------|------|
| **随机事件** | 每回合按概率触发 | 天灾、流言、意外等 |
| **条件触发事件** | 满足特定条件触发 | 如声望达标、官职变动、忠诚度阈值等 |
| **季节事件** | 特定季节触发 | 与季节弱影响机制配合 |
| **党争事件** | 党派行动触发 | 策反、弹劾、攻防等 |
| **事件链** | 前置事件结果影响后续事件 | 形成叙事连贯性 |

### 事件池（分段设计）

| 阶段 | 说明 |
|------|------|
| **地方官阶段** | 地方治理、民生、考核相关事件池 |
| **中枢阶段** | 朝堂博弈、党争、圣眷相关事件池 |
| **党派领袖阶段** | 党派运营、大规模攻防相关事件池 |

### 事件稀有度

| 稀有度 | 说明 |
|--------|------|
| **普通** | 高概率触发，常规影响 |
| **稀有** | 低概率触发，较大影响/特殊奖励 |
| **史诗** | 极低概率触发，重大影响/关键奖励 |

### 抉择机制：选择+判定混合

```
事件发生 → 玩家选择分支 → 角色属性/资源/条件判定 → 确定最终结果
```

### 影响范围

- **多维度影响**：资源、声望、官职、忠诚、稳固值、政绩槽等均可被事件影响
- **事件链**：当前事件的结果会影响后续事件的发生条件和内容

## 十、系统八：剧情/对话系统

### 对话系统

- **线性对话为主**：对话以叙事推进为主，无复杂分支网
- **弱联动**：对话选择偶尔影响系统数值，主要推动叙事

### 剧情推进：混合解锁

| 解锁方式 | 说明 |
|----------|------|
| **条件解锁** | 满足声望/官职/资源等条件触发新剧情 |
| **时间解锁** | 时间推进到特定节点自动触发 |
| **任务解锁** | 完成特定任务后触发对应剧情 |

### 主线剧情

> 主线剧情设计详见独立文档：[argue_game_diag.md](argue_game_diag.md)
>
> 概要：主角13岁开局，经历新手教学→灭门之灾→科举入仕→地方晋阶→中枢博弈→复仇终局6期，全局约30年，最终在42岁前完成复仇。

## 十一、系统九：任务系统

### 任务类型

| 类型 | 说明 |
|------|------|
| **主线任务** | 推动复仇剧情，有期限，失败则游戏结束 |
| **支线任务** | 额外奖励/拓展内容，有期限，失败仅丢失奖励 |
| **党派任务** | 党争相关任务，有期限，失败丢失奖励 |

### 任务奖励（混合）

| 奖励类型 | 示例 |
|----------|------|
| **资源奖励** | 资源点、金钱、声望、党派贡献等 |
| **解锁奖励** | 新剧情、新功能、新角色卡、新建筑等 |

### 失败机制

- **所有任务都有期限**
- 主线任务失败 → 游戏结束
- 支线/党派任务失败 → 仅丢失奖励，无额外惩罚

## 十二、系统间关联总览

```
时间流逝系统（骨架）
    ↓ 每回合推进 + 特殊节点(述职/大朝会/科举)
资源系统 ← 家族系统（建筑产出资源点 + 俸禄）
    ↓ 资源分配        ↓ 角色卡放入建筑      ↓ 家族行动(联姻/结交)
人物/角色系统 → 官场系统（能力影响政绩槽/稳固值）
    ↓ 传承/获取/移除         ↓ 地方官政绩槽 → 中枢稳固值
党争系统 ←——————→ 官场系统（攻防稳固值/党派升迁）
    ↓ 党派机构/行动/忠诚
事件系统（多维度影响所有系统，分段事件池，稀有度）
    ↓
剧情/对话系统（混合解锁推进，多结局）
    ↓
任务系统（驱动目标与奖励，期限机制）
    ↓
新游戏+（通关继承重开）
```
