# 官场党争复仇游戏 — 需求设计文档

---

## 一、核心定位

- **类型**：回合制策略养成
- **主题**：官场党争 + 复仇主线
- **核心体验**：在朝堂权谋中求生、攀爬、复仇——每一步选择都有代价
- **多结局设计**：根据玩家选择和成就有多个结局（复仇成功/失败等）
- **失败重开**：游戏失败后可继承声望/金钱获取加成重新开始

---

| # | 系统 | 核心职责 |
|---|------|----------|

## 三、系统一：时间流逝系统

### 基础规则

| 要素 | 设计 |
|------|------|
| **基本单位** | 1回合 = 1月 |
| **时间层级** | 月 → 季(3月) → 年(4季) |
| **回合流程** | 决策阶段 → 结算阶段 → 事件触发 → 下一回合 |
| **行动与回合** | 资源点限制行动，耗尽则无法行动，自动进入结算阶段；不可手动跳过回合 |
| **季节影响** | 弱影响——仅影响事件池中部分事件的触发权重（±5%），不改变行动消耗或产出公式 |
| **时间压力** | 软压力——各阶段有期限，超时则触发失败结局（见剧情文档）；日常拖延不额外惩罚 |
| **全局跨度** | 约30年（360回合） |

### 特殊时间节点

| 节点 | 周期 | 说明 |
|------|------|------|
| **年度述职** | 每年 | 地方官政绩槽评级，决定升迁/罢免 |
| **大朝会** | 稳固值归零时触发（从二品以上） | 从二品以上中枢官员弹劾辩论，决定官员命运 |
| **科举** | 每3年 | 家族成员参加，按宋制三试流程 |

### 大朝会机制

#### 触发条件与参与方

| 要素 | 设计 |
|------|------|
| **触发条件** | 中枢从二品以上官员稳固值降为0 |
| **攻击方** | 造成稳固值归零的党派，该党派中枢所有官员参与 |
| **防守方** | 稳固值归零的官员所在党派，该党派中枢所有官员参与 |
| **攻击方主控** | 造成稳固值最后伤害的官员自动担任主攻方，操控连线实体 |
| **防守方主控** | 稳固值归零的官员自动担任主守方，操控防御技能 |

#### 大朝会结果

| 结果 | 条件 | 效果 |
|------|------|------|
| **防守方被击倒** | 主攻方在时限内将所有逻辑链路连接成功 | 防守方**罢免**，失去官职 |
| **防守方未被击倒** | 时限结束，逻辑链路未全部连接 | 防守方**贬官**（降一阶） |
| **反击成功** | 防守方将主攻方实体血量降为0 | 防守方**保官**，获得3回合稳固值不可攻击buff |

#### 连线小游戏规则

**基本信息：**

| 要素 | 设计 |
|------|------|
| **游戏类型** | 连线小游戏——主攻方操作实体在场地上自由探索，寻找并连接逻辑链路 |
| **总时限** | 60秒 |
| **逻辑链路数量** | 随防守方品级变化：从二品4条、正二品5条、从一品6条、正一品7条 |
| **胜利条件（攻方）** | 在时限内将全部逻辑链路连接 → 击倒防守方 |
| **胜利条件（守方）** | 消灭主攻方实体 → 反击成功；或时限结束未全部连接 → 防守方存活（贬官） |

**主攻方实体：**

| 要素 | 设计 |
|------|------|
| **血量** | 口才 × 3 + 学识（上限300） |
| **移动** | 自由移动，躲避防守方技能 |
| **目标** | 接近逻辑链路节点并连接，每连接1条推进1步，全部连接则击倒防守方 |
| **被消灭** | 血量归零 → 攻击方失败，防守方反击成功保官 |

**防守方技能：**

| 技能类型 | 技能名 | 效果 | 冷却 | 消耗 |
|----------|--------|------|------|------|
| **常规·障碍** | 设障 | 在场地上放置1个障碍物，阻挡主攻方移动路径 | 基础10秒 | 无 |
| **常规·弹幕** | 射弹 | 向主攻方实体发射1枚弹幕，造成5点伤害 | 基础4秒 | 无 |
| **圣眷·伤害** | 雷霆 | 对主攻方实体造成40点伤害 | 无 | 5圣眷 |
| **圣眷·清链** | 断链 | 断开1条已连接的逻辑链路 | 无 | 8圣眷 |
| **圣眷·范围控制** | 天罗 | 全屏放置障碍物 + 3秒内弹幕伤害×2 | 无 | 10圣眷 |

- 常规技能冷却受防守方口才影响：实际冷却 = 基础冷却 × (1 - 口才/300)，最低冷却不低于基础冷却的50%
- 圣眷技能仅消耗防守方家族自身的圣眷，不可由党派其他成员捐献
- 若防守方家族圣眷为0，则无法使用任何圣眷技能，仅能依靠常规技能（障碍/弹幕）防御

#### 主角参与规则

| 主角身份 | 操作内容 |
|----------|----------|
| **主角为主攻方** | 操作连线实体，自由移动、躲避技能、连接链路 |
| **主角为防守方** | 操作防御技能，释放障碍/弹幕/圣眷技能 |
| **主角不参与** | 不进入小游戏，自动结算 |

#### NPC大朝会自动结算

当主角不参与大朝会时，结果自动计算：

```
击倒概率 = min(0.95, 主攻方口才 / max(1, 主攻方口才 + 守方口才) × (1 + 圣眷差加成))
  - 圣眷差加成 = max(0, (攻方党派总圣眷 - 守方党派总圣眷) / 100 × 0.05)，上限+0.3
  - 党派总圣眷 = 该党派所有中枢官员所在家族的圣眷之和

反击概率 = min(0.5, 守方口才 / max(1, 主攻方口才 + 守方口才) × (守方圣眷 / (守方圣眷 + 10)) × 0.5)

未击倒概率 = max(0.05, 1 - 击倒概率 - 反击概率)
```

- 击倒概率上限0.95，反击概率上限0.5，未击倒概率下限0.05，确保概率值始终合法
- max(1, ...) 防止双方口才均为0时除零

## 四、系统二：资源系统

### 核心资源模型

| 资源 | 获取 | 消耗/机制 |
|------|------|-----------|
| **资源点** | 每月固定刷新（基础3点）；声望每+1000增加1点，上限10点/月 | 投入建筑产出金钱/声望；行动消耗资源点 |
| **金钱** | 建筑产出；俸禄（按月发放）；事件/任务奖励；私库分红 | 维护声望（抵消衰减）；联姻；结交；其他行动消耗 |
| **声望** | 建筑产出；事件/任务奖励 | 每月衰减（固定+阶梯）；达到阈值提升资源点刷新量；需消耗金钱维护 |
| **圣眷** 🔒 | 步入中枢后解锁；资源点投入建筑产出；官职生产；任务/事件；与皇族关系 | 每月百分比衰减；声望和官职越高衰减越快 |
| **戒心** 🔒 | 步入中枢后解锁；圣眷衰减时转化而来 | 获取圣眷可降低戒心；戒心过高→家族debuff + 概率触发贬官/罢免事件 |

> 🔒 = 步入中枢（正三品以上）后解锁

### 核心循环

```
资源点(每月刷新，详见下方)
    ↓ 投入建筑
建筑产出 = (建筑基础产出 + 角色能力 × 建筑基础系数 × 等级倍率) × 资源点加成系数
    ↓ 产出
金钱 / 声望 / 圣眷

声望 → 提升资源点刷新量(每+1000声望→+1点/月) → 更多资源点可用
  ↓
声望衰减(固定+阶梯) ← 需金钱维护(1贯抵1点) ← 金钱

[中枢后]
资源点投入建筑 → 圣眷(百分比衰减) + 戒心(圣眷衰减转化)
圣眷衰减 → ×0.5 → 戒心增长
获取圣眷 → ×0.3 → 降低戒心
高声望/高官 → 加剧圣眷衰减与戒心增长 → 需要平衡经营
戒心过高 → 家族debuff + 概率触发贬官/罢免事件
```

### 资源点

#### 刷新规则

| 要素 | 设计 |
|------|------|
| **基础刷新** | 每月3点 |
| **声望加成** | 声望每增加1000，刷新量+1点 |
| **上限** | 10点/月 |
| **刷新时机** | 每月（回合开始时）刷新，未用完的资源点不累积（不跨月） |
| **用途** | 投入建筑产出金钱/声望/圣眷；执行各种行动消耗 |

#### 建筑产出公式

> 建筑分为家族建筑与城市建筑两大类。家族建筑由玩家建造，产出声望/圣眷/角色成长；城市建筑为州城固有设施，玩家通过派遣角色卡入驻获取金钱和声望。详见 [argue_game_architecture.md](argue_game_architecture.md)

**家族建筑产出：**

```
资源产出 = (建筑基础产出 + 角色对应能力 × 建筑基础系数 × 等级倍率) × 资源点加成系数
成长产出 = 基础成长速率 × 建筑基础系数 × 等级倍率 × 潜力值 × 年龄段速率 × buff加成
```

**城市建筑产出：**

```
金钱产出 = (建筑基础产出 + 角色商业 × 建筑基础系数 × 等级倍率) × 资源点加成系数 × (1 + 司户参军加成) × (1 + 州城繁荣度加成)
  其中：司户参军加成 = 家族成员在该州担任司户参军时为+20%(即×1.2)，否则为0
声望产出 = (建筑基础产出 + 角色口才 × 建筑基础系数 × 等级倍率) × 资源点加成系数 × (1 + 州城繁荣度加成)
成长产出 = 基础成长速率 × 建筑基础系数 × 等级倍率 × 潜力值 × 年龄段速率 × buff加成
```

- 城市建筑为州城固有公共设施，角色入驻金钱类建筑无消耗；入驻声望类/成长类建筑每回合需消耗金钱

| 投入资源点数 | 加成系数 |
|-------------|---------|
| 0点 | 100%（不投入，仅靠角色基础产出） |
| 1点 | 150% |
| 2点 | 200% |
| 3点 | 250%（上限） |

- 每个建筑每回合最多投入3点资源点
- 角色对应能力：声望建筑看学识（家族建筑）/口才（城市建筑），圣眷建筑看口才，金钱建筑看商业，成长建筑看对应能力
- 建筑基础产出和建筑等级系数详见 [argue_game_architecture.md](argue_game_architecture.md)

#### 行动消耗参考

| 行动 | 消耗资源点 | 说明 |
|------|-----------|------|
| 建筑投入 | 1~3点/建筑 | 每月投入，产出金钱/声望/圣眷 |
| 结交 | 1~3点 | 派出角色结交其他家族人物，另需投入金钱和声望 |
| 联姻 | 1~3点 | 发起联姻消耗，另需投入金钱和声望 |
| 攻防稳固值 | 攻讦1~3点/检举1~3点+情报/离间1~3点+情报/弹劾3~5点+10圣眷+1张弹劾情报 | 中枢攻防消耗 |
| 稳固值防御 | 1~3点+0~30圣眷 | 月初预设，减伤10%~80% |
| 稳固值提升 | 3圣眷或50声望/点 | 未被攻击回合可用，每回合最多10点 |
| 商议 | 1~2点 | 转租/出让/分成城市建筑槽位 |
| 拉拢策反 | 1~3资源点+金钱 | 降低目标家族党派忠诚度：忠诚度降低值 = 投入资源点×3 + 投入金钱/100(向下取整)；目标忠诚度≤30时可直接策反(转入党派) |

### 金钱

#### 单位

- **文**：基础单位
- **贯**：1贯 = 1000文

#### 获取

| 来源 | 说明 |
|------|------|
| **建筑产出** | 投入资源点到金钱类建筑，按建筑产出公式计算（详见建筑系统设计文档） |
| **俸禄** | 官员按月领取，散官（有品阶无官职）同样领取俸禄 |
| **事件/任务奖励** | 随机获取 |
| **私库分红** | 党派私库年度分红 |

| 正一品 | 100 | 从五品 | 50 |
| 从一品 | 94 | 正六品 | 44 |
| 正二品 | 89 | 从六品 | 39 |
| 从二品 | 83 | 正七品 | 33 |
| 正三品 | 78 | 从七品 | 27 |
| 从三品 | 72 | 正八品 | 22 |
| 正四品 | 67 | 从八品 | 16 |
| 从四品 | 61 | 正九品 | 11 |
| 正五品 | 55 | 从九品 | 5 |

> 注：品阶系数为线性，具体数值可在平衡阶段微调

#### 消耗

| 用途 | 消耗 | 说明 |
|------|------|------|
| **维护声望** | 1贯抵消1点声望衰减 | 可全额抵消，无上限 |
| **联姻** | 无上限 | 作为影响联姻成功率的维度之一，投入越多成功率越高 |
| **结交** | 以文为单位投入，取对数衰减 | 投入越多关系增量越大，但边际递减 |
| **城市建筑入驻** | 声望类/成长类每回合消耗 | 金钱类建筑无需消耗 |
| **拉拢策反** | 金钱+资源点 | 降低目标家族党派忠诚度 |

### 声望

#### 数值范围

- 无上限

#### 获取

| 来源 | 说明 |
|------|------|
| **建筑产出** | 投入资源点到声望建筑，按建筑产出公式计算 |
| **官职声望加成** | 担任官职每月自动获得声望，品级越高加成越多（见下表） |
| **事件/任务奖励** | 随机获取 |

#### 官职声望加成（按月发放）

- 仅在任官员享受声望加成，散官不享受
- 品阶越高加成增长越快（指数曲线），体现中枢权力带来的声望收益

| 品阶 | 声望加成（/月） | 品阶 | 声望加成（/月） |
|------|----------------|------|----------------|
| 正一品 | 500 | 从五品 | 45 |
| 从一品 | 305 | 正六品 | 35 |
| 正二品 | 230 | 从六品 | 25 |
| 从二品 | 175 | 正七品 | 20 |
| 正三品 | 130 | 从七品 | 15 |
| 从三品 | 100 | 正八品 | 10 |
| 正四品 | 75 | 从八品 | 10 |
| 从四品 | 65 | 正九品 | 5 |
| 正五品 | 60 | 从九品 | 5 |

#### 衰减公式

采用**固定+阶梯**模型：

```
每月衰减量 = 3 + floor(当前声望 / 100) × 3
```

- 基础每月衰减3点
- 当前声望每增加100点，额外衰减3点

| 当前声望 | 每月衰减 |
|----------|-----------|
| 0~99 | 3点 |
| 100~199 | 6点 |
| 500~599 | 18点 |
| 1000~1099 | 33点 |
| 5000~5099 | 153点 |
| 10000~10099 | 303点 |

#### 金钱维护

- 每消耗1贯金钱可抵消1点声望衰减
- 可全额抵消，无上限
- 维护时机：每月结算阶段，先计算衰减量，再根据投入金钱抵消

#### 声望阈值与资源点加成

| 声望区间 | 资源点刷新加成 |
|----------|--------------|
| 0~999 | +0点（基础3点） |
| 1000~1999 | +1点（共4点） |
| 2000~2999 | +2点（共5点） |
| 3000~3999 | +3点（共6点） |
| 4000~4999 | +4点（共7点） |
| 5000~5999 | +5点（共8点） |
| 6000~6999 | +6点（共9点） |
| 7000+ | +7点（共10点，达到上限） |

- 资源点刷新上限为10点/月（声望达到7000即可满额）

#### 声望与家族等级

| 家族等级 | 声望条件 | 人数上限 |
|----------|----------|----------|
| **初级家族** | 初始 | 10人 |
| **中级家族** | 声望≥3000 | 20人 |
| **顶级家族** | 声望≥7000 | 30人 |

#### 延迟生效规则

所有由资源/等级推导的派生属性，均在**回合结束时结算，下月生效**，当月不立即变动：

| 派生属性 | 推导来源 | 生效时机 |
|----------|----------|----------|
| 家族等级 | 家族声望 | 回合结束时判定，下月生效 |
| 资源点上限 | 家族声望 | 回合结束时判定，下月生效 |
| 建筑产出 | 建筑等级/角色能力/繁荣度 | 回合结束时判定，下月生效 |
| 城市建筑等级 | 繁荣度+知州升级 | 回合结束时判定，下月生效 |

- 例：本月声望从3100降至2800，家族等级仍按中级运作，回合结束时判定降级，**下月**按初级运作
- 例：本月投入声望进行结交，声望扣除后资源点上限当月不变，**下月**按新声望值刷新

### 圣眷 🔒

> 步入中枢（正三品以上）后解锁

#### 数值范围

- 无上限

#### 获取

| 来源 | 说明 |
|------|------|
| **建筑产出** | 投入资源点到圣眷类建筑，按建筑产出公式计算 |
| **官职生产** | 中枢官职每月自动产出圣眷，品级越高产出越多 |
| **任务/事件** | 特定任务和事件奖励 |
| **皇族关系** | 与皇族（皇族也是一个家族）关系越高，获取圣眷的渠道越多 |

#### 官职圣眷产出

| 品阶 | 每月圣眷产出 |
|------|-------------|
| 正三品 | 1 |
| 从二品 | 3 |
| 正二品 | 7 |
| 从一品 | 15 |
| 正一品 | 30 |

```
产出公式 = round(0.4 × 2^(品阶序号))
品阶序号：正三品=1, 从二品=2, 正二品=3, 从一品=4, 正一品=5
```

#### 衰减公式

采用**百分比衰减 + 品阶系数倍率**模型：

```
每月衰减量 = 当前圣眷 × 1% × 品阶衰减系数 × 声望衰减加成
```

**品阶衰减系数：**

| 品阶 | 系数 |
|------|------|
| 正三品 | 1.0 |
| 从二品 | 1.2 |
| 正二品 | 1.4 |
| 从一品 | 1.6 |
| 正一品 | 1.8 |

- 每升一阶（从正三品起），系数+0.2

**声望衰减加成：**

- 声望超过10000时，每增加1000声望，衰减系数+0.1
- 公式：`声望衰减加成 = 1 + max(0, (声望 - 10000) / 1000) × 0.1`

#### 衰减示例

| 品阶 | 声望 | 当前圣眷 | 品阶系数 | 声望加成 | 每回合衰减 |
|------|------|---------|---------|---------|-----------|
| 正三品 | 5000 | 200 | 1.0 | 1.0 | 2.0 |
| 正二品 | 8000 | 500 | 1.4 | 1.0 | 7.0 |
| 正一品 | 15000 | 1000 | 1.8 | 1.5 | 27.0 |

### 戒心 🔒

> 步入中枢（正三品以上）后解锁

#### 数值范围

- 0~100

#### 增长机制

**核心公式：圣眷衰减→戒心转化**

```
戒心增长 = 圣眷本月衰减量 × 0.5
```

- 圣眷每月衰减时，衰减量的50%转化为戒心
- 无自然衰减，戒心不会自行降低

#### 降低机制

**获取圣眷→降低戒心**

```
戒心降低 = 本月获取的圣眷 × 0.3
```

- 获取圣眷可降低戒心，比例为1:0.3
- 戒心最低为0

#### 每回合戒心变化公式

```
第一步：戒心 += 圣眷衰减量 × 0.5（衰减转化）
第二步：戒心 -= 本月获取圣眷 × 0.3（获取降低）
```

- 两步分开计算，戒心最低为0

#### debuff阈值与效果

| 戒心区间 | 等级 | 效果 |
|----------|------|------|
| 0~29 | 安全 | 无debuff |
| 30~59 | 警戒 | 俸禄-10%，官职声望加成-10%，官职圣眷产出-10% |
| 60~79 | 危险 | 俸禄-20%，官职声望加成-20%，官职圣眷产出-20%，家族所有官职稳固值被攻击时受伤系数×1.2 |
| 80~100 | 极危 | 俸禄-30%，官职声望加成-30%，官职圣眷产出-30%，家族所有官职稳固值被攻击时受伤系数×2.0 |

#### 贬官/罢免触发概率

| 戒心 | 触发概率 |
|------|---------|
| 80~89 | 每月5%概率触发贬官/罢免事件 |
| 90~94 | 每月15%概率触发 |
| 95~99 | 每月30%概率触发 |
| 100 | 必定触发贬官/罢免事件 |

#### 贬官/罢免执行规则

- 事件触发时，罢免/贬官对象为**家族内官职最高者**
- 家族内多人同品阶时，优先罢免/贬官资历较低者

**戒心降低规则：**

| 处置结果 | 戒心降低量 |
|----------|-----------|
| 每贬低1阶 | 降低5戒心 |
| 免官（完全罢免） | 降低30戒心 |
| 降低上限 | 单次事件最多降低30戒心 |

- 贬官幅度越大，戒心降低越多，但单次最多降30
- 被贬低阶数从当前品阶计算，最低贬至从九品
- 被免官者削职为民——失去品阶和官职，不再领取俸禄，可重新通过科举入仕

### 关键设计点

- **核心决策**：每回合有限的资源点如何分配（哪个建筑投入？投入几点？金钱/声望/圣眷如何取舍？）
- **声望的马太效应与衰减**：高声望带来更多资源点，但衰减更快、维护成本更高
- **金钱维护声望**：1贯抵1点衰减，全额可抵消，但金钱还有其他用途——是否全花在维护上是关键抉择
- **圣眷与戒心的博弈**：圣眷衰减→戒心增长，获取圣眷→降低戒心，形成"越强越危险"的核心张力
- **中枢平衡术**：高品阶高声望加剧圣眷衰减和戒心增长，需要持续经营才能维持权力

### 国库系统

#### 基本规则

- 国库是一个全局数值，代表国家财政储备
- 国库资金来源：各州县税收上缴部分
- 国库资金用途：城市建筑升级支出、国家事件支出等

#### 党派国库上缴对比

- 每季度结算时，统计畿党与淮党所属州县的税收上缴总额
- 上缴更多的党派，其中枢官员所在家族获得额外圣眷奖励

**圣眷换算规则：**

```
两党上缴差额 = |畿党上缴总额 - 淮党上缴总额|
胜出方圣眷奖励 = 差额 × 0.3（每100贯差额→30点圣眷）
```

- 奖励分配：胜出方所有在中枢有官员的家族，按官员品级权重分配圣眷奖励
- 品级权重：正一品×5, 从一品×4, 正二品×3, 从二品×2, 正三品×1
- 某家族所得圣眷 = 胜出方总奖励 × (该家族中枢官员品级权重之和 / 胜出方所有家族品级权重之和)

## 五、系统三：家族系统

### 核心模型

| 要素 | 设计 |
|------|------|
| **角色卡** | 每个角色是一张卡，有属性、能力，可被放置到建筑中 |
| **建筑系统** | 家族拥有各类建筑，角色卡放入建筑后产出对应收益；建筑种类后续再定 |
| **族长权限** | 族长可操控所有角色卡的放置；非族长成员只能操控自己的卡；只有族长有权驱逐族人（驱逐消耗2资源点） |
| **主控角色** | 玩家操控的角色，可能不是族长（需争取族长之位） |
| **族内关系图谱** | 清晰的族谱/关系网展示，无数值化关系值（纯图谱展示） |
| **资源归属** | 资源点、金钱、声望等均为家族层级拥有，非角色个人 |

### 家族人数上限

家族等级与人数上限见资源系统→声望与家族等级

- 人数到达上限后：适龄夫妻无法生育、无法嫁入人、不会有人来投奔

### 流浪池机制

- 被族长驱逐的族人进入流浪池，不直接视为死亡
- 流浪池中的人每月随机找一个家族请求收留
- 被拒绝后永远不会再次请求该家族
- 家族可在流浪池中主动查看并招募被驱逐的族人
- 族人在流浪池中呆满4个月未被收留则死亡

### 族长更替：竞争继承

族长死亡时，按以下优先级判定继承者：

**有官职者：**

1. **官位品级最高者**继承族长
2. 品级一致时，比拼**资历**，资历高者胜
3. 资历一致时，比拼**学识+口才+军事**三项能力值总和，高者胜

**族内无人做官：**

- 按**年龄从大到小**的顺序寻找：
  1. 同年龄者中**学识+口才+军事**三项能力值总和最高者继承

- 非族长的主控角色需争取族长之位

### 家族间关系

```
家族间关系 = min(100, 基础家族关系 + 成员关系加权平均值)
```

**成员关系加权平均值：**

双方家族所有成年角色（15岁及以上）与对面家族关系的加权平均值。

| 角色身份 | 权重 |
|----------|------|
| **族长** | 3 |
| **正三品以上官职** | 3 |
| **从四品~从三品官职** | 2 |
| **从九品~正四品官职** | 1.5 |
| **其余成年族人** | 1 |

- 仅计算15岁及以上的成年角色
- 族长和官职越高的人对家族间关系影响越大

**基础家族关系：**

- 初始为0，联姻成功后增加
- 基础家族关系是家族间关系的"护盾"——只要基础关系>0，对方家族成员对本家族的角色卡关系不会自然衰减
- 基础家族关系归零后，所有成员关系才进入衰减

**基础家族关系衰减公式：**

```
每月衰减量 = 声望差衰减 + 党派衰减（衰减至0为止，不降为负值）

声望差衰减 = ceil(max(0, 对方声望 - 主控声望) / 2000) × 0.5
  - 对方声望不高于主控时，声望差衰减 = 0（不下降）
  - 对方声望每高出2000，月衰减+0.5点

党派衰减 = 双方分属不同党派 ? 1 : 0
  - 同党派或无党派：0
  - 不同党派：每月衰减1点
```

**衰减示例：**

| 场景 | 对方声望 | 主控声望 | 声望差衰减 | 党派衰减 | 月衰减 |
|------|---------|---------|-----------|---------|--------|
| 同党,声望相近 | 3000 | 2800 | 0 | 0 | 0 |
| 同党,对方略高 | 5000 | 3000 | ceil(2000/2000)×0.5=0.5 | 0 | 0.5 |
| 同党,对方高 | 7000 | 1000 | ceil(6000/2000)×0.5=1.5 | 0 | 1.5 |
| 不同党,声望相近 | 3000 | 3000 | 0 | 1 | 1 |
| 不同党,对方高 | 6000 | 1000 | ceil(5000/2000)×0.5=1.5 | 1 | 2.5 |

- 联姻增加的基础关系通常在30~60区间
- 同党声望相近时基础关系几乎不衰减，联姻收益持久
- 不同党派联姻：月衰减1~2.5点，30点基础关系需12~30个月才归零
- 声望差距越大衰减越快，但整体速率远低于角色卡关系衰减

**基础关系护盾规则：**

| 基础家族关系状态 | 对方成员角色卡关系变化 |
|-----------------|---------------------|
| > 0 | 不自然衰减（结交可正常提升） |
| = 0 | 所有成员关系按比例衰减（见下方公式） |

**基础关系归零后的成员关系衰减：**

```
每月衰减 = 向0方向靠拢 ceil(当前关系绝对值 × 5%)
  - 正值每月减少5%
  - 负值每月增加5%
  - 最低衰减1点
  - 结交仍可正常提升，但衰减持续进行
```

- 基础关系归零前，联姻建立的"亲缘纽带"保护所有成员关系不被遗忘
- 基础关系归零后，失去亲缘纽带，关系开始淡忘——5%比例衰减使高关系值衰减快、低关系值衰减慢，符合"疏远从密切开始"的逻辑
- 重新联姻可恢复基础关系>0，立即停止成员关系衰减

### NPC角色卡关系衰减

NPC角色卡与主控家族的关系值受基础家族关系保护：

| 基础家族关系状态 | NPC角色卡关系变化 |
|-----------------|-----------------|
| > 0 | 不自然衰减（结交可正常提升） |
| = 0 | 每月向0方向靠拢1点（正值每月-1，负值每月+1） |

- 基础家族关系是"亲缘纽带"，只要纽带存在，成员间关系就受到保护
- 基础关系归零后，NPC角色卡关系每月衰减1点（固定值，非比例——单对单关系衰减较慢）
- 结交可提升关系值，事件也可能改变关系值
- 关系值影响：家族间关系计算、NPC家族在党争中的配合意愿

### 家族行动

#### 联姻

- 当家族内有未婚适龄角色时，可向有未婚异性角色的家族发起联姻
- 联姻需付出金钱和资源点

**联姻成功率公式：**

```
联姻判定值 = 投入资源点 × (投入声望 + 投入金钱(贯)) × max(1, 被联姻对象与发起家族关系 - 50)
           - rand(0.8, 1.2) × 联姻类型修正值 × (被联姻方家族声望 - 发起方家族声望 + 500)

结果：判定值 > 0 → 联姻成功；判定值 ≤ 0 → 联姻失败
```

- `被联姻对象与发起家族关系`：NPC角色卡与发起家族的关系值，-100~+100
- `max(1, 关系-50)`：关系需超过50才产生正向加成，低于50则基数仅为1
- `rand(0.8, 1.2)`：随机因子，增加不确定性
- 声望差+500为基础差值：对方声望越高越难联姻，即使双方声望相同也有500的基础难度

**联姻类型修正值：**

| 类型 | 说明 | 角色卡归属 | 修正值 |
|------|------|------------|--------|
| **婚娶** | 娶对方家族女性入本族 | 对方角色卡加入本家族 | 1.4 |
| **招婿** | 招对方家族男性入本族 | 对方角色卡加入本家族 | 1.4 |
| **婚嫁** | 本族女性嫁入对方家族 | 本族角色卡加入对方家族 | 0.8 |
| **入赘** | 本族男性入赘对方家族 | 本族角色卡加入对方家族 | 0.8 |

- 婚娶/招婿修正值1.4：将对方角色卡拉入本族，阻力更大
- 婚嫁/入赘修正值0.8：本族角色卡嫁出，阻力更小
- 修正值作用于减项——修正值越大，减项越大，联姻越难
- 联姻失败无资源损失，但5回合内不可再向该家族发起联姻

**联姻成功后基础家族关系增加公式：**

```
基础关系增量 = 15 + min(10, 联姻投入资源点 × 3) + min(5, floor(投入声望 / 100))

上限：单次联姻最多增加30点基础家族关系
```

- 基础值15：联姻成功即获得15点基础关系
- 资源点加成：每点资源点+3，最多+10（投入4点即满）
- 声望加成：每投入100点声望+1，最多+5（投入500点声望即满）
- 多次联姻叠加：同一对家族可多次联姻，基础关系增量累加，上限100

#### 结交

- 每回合可花费资源点和金钱对其他家族中的人物进行结交
- 需派出家族内一名角色卡执行结交
- 结交增加目标人物对本家族的关系值，结果向下取整

**结交关系增加公式：**

```
关系增量 = max(1, ceil(声望系数 × (ln(投入金钱(文) + 1) + ln(投入声望 + 1) × 3) × 投入资源点 × (派出角色口才 / 100)))

声望系数 = min(max(0.1, 发起家族声望 / 被结交家族声望), 10)
关系增量上限 = 25（每次结交最多增加25点关系值）
```

- `max(1, ...)`：最低增加1点关系值（即使投入极低也有效果）
- `ceil`：向上取整
- `声望系数`：发起方声望越高越容易结交，但最低0.1（即使声望远低于对方仍可尝试），最高10（声望碾压也不会无限放大）
- `ln(投入金钱(文) + 1)`：金钱投入取对数衰减，大额投入边际递减
- `ln(投入声望 + 1) × 3`：声望投入取对数衰减后×3，声望比金钱更有效。投入的声望从家族声望池中扣除
- `投入资源点`：资源点直接作为乘数
- `口才/100`：口才100时系数为1，口才50时系数为0.5，口才直接影响结交效率
- **关系增量上限25**：每次结交最多增加25点关系值
- **声望扣除**：投入的声望从家族声望池中实际扣除，扣除后当月声望衰减仍按扣除后的值计算

#### 商议

- 当城市建筑槽位被其他家族占用时，可发起商议获取槽位使用权
- 需派出家族内一名角色卡执行商议
- 三种商议类型：

| 类型 | 说明 | 效果 |
|------|------|------|
| **转租** | 支付金钱/资源点租用槽位3年 | 租期内产出归我方，到期归还 |
| **出让** | 支付大量金钱+声望买断 | 对方永久撤出，产出永久归我方 |
| **分成** | 协商共享该槽位产出 | 无需角色卡入驻，投入资源点获取30%~50%产出分成 |

- 建筑类型约束：成长类建筑（官学/武馆/茶楼）不可转租和分成，必须出让；特色建筑成功率额外降低
- 商议成功率受商议类型、建筑类型与产出占比、家族关系、投入金钱/声望/资源点、双方声望差距、己方官职影响
- 详见建筑系统设计文档 → 商议机制

### 核心玩法循环

```
角色卡(属性/能力) → 放入建筑 → 产出收益(资源点/金钱等)
                                    ↓
族长统筹所有卡的放置策略 ← 资源回流到家族
玩家若非族长 → 只能操控自己的卡 → 需争取族长权限

家族行动：联姻(获取角色卡) / 结交(提升关系) / 商议(城市建筑槽位竞争) / 情报升级(暗阁提升情报等级)
```

### 能力值与潜力值系统

#### 潜力值公式

```
潜力值 = (rand(父能力值, 母能力值) / 100 + 1) / 2
```

- 先在父母该能力值之间随机取一个值，再代入公式计算潜力
- 潜力值范围：0.5~1.0（当随机值为0时最低0.5，随机值为100时最高1.0）
- 潜力值在出生瞬间确定，终身不可更改

**举例：** 父学识80，母学识60 → rand(60,80) 如得72 → 潜力=(72/100+1)/2=0.86

#### 潜力值作用

- **潜力值 = 成长速率**：能力上限仍为100
- 潜力值越高，能力成长越快
- 角色刚出生时所有能力值初始为0

#### 能力成长公式

```
每月成长量 = 建筑物基础速率 × 潜力值 × 年龄段速率 × buff加成
```

#### 年龄段速率

| 年龄段 | 速率 | 说明 |
|--------|------|------|
| **1-10岁** | 200% | 孩童期，成长最快 |
| **10-18岁** | 100% | 少年期，正常成长 |
| **18-25岁** | 50% | 青年期，成长放缓 |
| **25-30岁** | 20% | 壮年期，成长缓慢 |
| **超过30岁** | 0% | 不再成长 |

#### 能力成长方式

| 方式 | 说明 |
|------|------|
| **建筑被动成长** | 角色卡放置在对应建筑中时，根据时间流逝自动成长 |
| **资源点增益** | 投入资源点可获得成长增益（buff加成），加速能力提升 |
| **学识成长小游戏** | 主控家族有角色卡在学识成长类建筑中时，每回合可进行限时填字小游戏，获得学识成长加成（详见下方） |

#### 学识成长：限时填字小游戏

**触发条件：** 主控家族有角色卡放置在学识成长类建筑（私塾）中时，每回合触发一次

**游戏规则：**

| 要素 | 设计 |
|------|------|
| **类型** | 限时填字——给出经典语句的上下文，空出关键字，玩家从候选字中选择填入 |
| **时间限制** | 20秒 |
| **题目来源** | 经史子集名句，与游戏世界观契合。题库分类框架如下： |
| **核心机制** | 20秒内填入越多正确字词，加成越高 |
| **跳过** | 玩家可选择跳过小游戏，跳过则默认获得80%的当月学识成长奖励 |

**难度与收益随学识递减：**

建筑内所有角色卡的学识平均值决定填字难度和单次收益：

| 平均学识 | 填字模式 | 候选字数 | 每成功填入1字加成 | 加成上限 |
|----------|----------|----------|-------------------|----------|
| 0-30 | 单字填充 | 4个 | +50% | 200% |
| 31-60 | 双字填充 | 5个 | +35% | 200% |
| 61-80 | 三字填充 | 6个 | +25% | 200% |
| 81-100 | 四字填充 | 7个 | +15% | 200% |

**题库分类框架：**

| 分类 | 内容 | 难度层 | 占比 |
|------|------|--------|------|
| **经** | 《论语》《孟子》《诗经》《尚书》等儒家经典 | 基础→进阶 | 40% |
| **史** | 《史记》《汉书》《资治通鉴》等史书名句 | 基础→进阶 | 25% |
| **子** | 《老子》《庄子》《韩非子》等诸子百家 | 进阶→高难 | 20% |
| **集** | 唐宋诗词名篇、赋文 | 基础→进阶 | 15% |

- 难度层与填字模式对应：基础=单字填充(学识0-30)、进阶=双字/三字填充(学识31-80)、高难=四字填充(学识81-100)
- 具体题目内容后期填充，每难度层至少准备50题以保证不重复

- 平均学识越高→单字加成越低、难度越高（需要填更多字、干扰项更多），但理论上限仍可达200%
- 实际收益：低学识时容易拿满加成；高学识时需要填更多字且每字收益更低，难以达到200%，符合"越学越难"的规律
- 填错不扣减，只是不加成
- 加成作用于该私塾中所有角色的当月学识成长

**加成计算示例：**

| 场景 | 平均学识 | 成功填入 | 加成计算 | 最终加成 |
|------|----------|----------|----------|----------|
| 新手私塾 | 15 | 填对2字 | 2 × 50% = 100% | 100% |
| 新手私塾 | 15 | 填对4字 | 4 × 50% = 200% | 200%（封顶） |
| 中阶私塾 | 50 | 填对4字 | 4 × 35% = 140% | 140% |
| 中阶私塾 | 50 | 填对6字 | 6 × 35% = 210% | 200%（封顶） |
| 高阶私塾 | 75 | 填对6字 | 6 × 25% = 150% | 150% |
| 顶级私塾 | 90 | 填对8字 | 8 × 15% = 120% | 120% |

### 特性系统

#### 特性等级

| 等级 | 生成概率 | 说明 |
|------|----------|------|
| **寻常** | 40% | 普通特性 |
| **清良** | 30% | 较好特性 |
| **翘楚** | 20% | 稀有特性 |
| **绝伦** | 10% | 顶级特性 |

- 每个等级有对应的特性库

#### 特性生成流程

角色生成时依次判定是否生成特性，最多4项：

```
第1个特性：60%概率生成 → 若生成则继续判定第2个
第2个特性：40%概率生成 → 若生成则继续判定第3个
第3个特性：20%概率生成 → 若生成则继续判定第4个
第4个特性：10%概率生成 → 结束
```

- 判定为"生成"后，再按四个等级的生成概率（寻常40%/清良30%/翘楚20%/绝伦10%）判定等级
- 判定等级后，从对应等级的特性库中随机抽取一个特性赋予角色

#### 特性继承（生育时判定，优先于随机生成）

- 父母双方的所有特性均有50%概率遗传给孩子
- 按特性等级从高到低依次判定（绝伦→翘楚→清良→寻常）
- 生成/继承时若出现互斥组合，保留等级较高的特性，丢弃等级较低的；等级相同则保留后判定的
- 满4个特性后不再判定
- 若父母所有特性遗传判定结束后孩子特性未满4个，再进行随机特性生成判定

#### 特性获取与移除

| 要素 | 设计 |
|------|------|
| **获取** | 父母特性继承（优先） / 初始随机生成 / 后期事件获得 |
| **移除** | 部分特性可通过特殊事件/条件移除（非全部） |
| **效果** | 每个特性对能力值产生修正 |

| 性别 | 可婚育年龄 | 说明 |

- 任何一方超出范围都无法生育

### 生育概率

- 男女结婚的那个月（回合）开始计算
- 每月进行一次怀孕判定
- 判定概率：首次20%，每成功怀孕一次后概率减少5%
- 判定为怀孕后冷却24个月（2年），期间不再进行怀孕判定
- 冷却结束后重新进入判定周期，概率按已怀孕次数递减

### 年龄与死亡系统

- 角色寿命范围：0-100岁
- 年龄随游戏时间推进增长
- 传承机制确保老角色退场后新角色接替
- 主角死亡 → 游戏结束

#### 死亡概率（每月判定一次）

|------|------|
| **年龄** | 15岁及以上 |
| **性别** | 无限制，男女同等条件参加 |
| **科举资格** | 未参加过科举者方可参加（散官已经参加过科举，不可再参加） |
| **学识** | 参加科举无学识门槛（解试需学识≥50方可通过） |
| **报名费** | 每试5贯（解试5贯，省试5贯，殿试5贯） |
| **建筑限制** | 参与科举的角色该回合无法放入任何建筑槽；已在建筑槽中的角色需退出1回合方可参加科举 |
| **报名时机** | 每试前1个月报名缴费 |
| **次数限制** | 每人一生最多参加3次科举（主角仅1次机会，不中即游戏结束） |

#### 科举时间线

```
科举年7月  → 报名解试（7月报名缴费，8月考试）
次年1月     → 报名省试（1月报名缴费，2月考试，当月出榜）
次年2月     → 报名殿试（省试出榜后当月报名缴费，3月考试）
```

- 省试当月出榜：2月考试结束后当月公布成绩，通过者当月即可报名殿试

#### 第一试：解试（州试）

**规则：**

- 在各州分别举行，按州独立判定
- 仅考察学识属性
- 门槛制：学识≥50即可通过，获解额
- 未通过者下次科举需重新参加解试

**判定方式：**

```
学识 ≥ 50 → 通过，获解额
学识 < 50 → 未通过
```

- 无名额上限，只要学识达标即通过

#### 隐藏分机制

省试与殿试的评分公式均附带一个**隐藏分**，玩家不可见。

**基本规则：**

- 每位考生每次科举时随机生成一个0.1~0.9的隐藏分（均匀随机）
- 隐藏分正常情况下**不参与**评分公式计算
- 仅在综合分相同时，比较隐藏分作为排名依据，隐藏分高者排前
- 隐藏分仍相同则随机决定

**礼部尚书操控：**

- 拥有**礼部尚书**官职的家族可操控科举隐藏分
- 该家族可将指定考生的隐藏分设为大于1的值（具体数值由家族投入资源决定，见下方公式）
- 当隐藏分 > 1时，该隐藏分**直接加入评分公式**参与计算

**操控消耗公式：**

```
操控消耗 = (目标隐藏分 - 1)² × 20 圣眷 + (目标隐藏分 - 1) × 100 声望

目标隐藏分上限 = 5.0
每次科举最多操控2名考生
```

**消耗示例：**

| 目标隐藏分 | 圣眷消耗 | 声望消耗 | 说明 |
|-----------|---------|---------|------|
| 1.5 | 5 | 50 | 小幅操控，性价比高 |
| 2.0 | 20 | 100 | 中等操控 |
| 3.0 | 80 | 200 | 大幅操控，圣眷消耗显著 |
| 5.0 | 320 | 400 | 极限操控，圣眷代价极大 |

- 二次方设计使小幅操控性价比高、大幅操控代价剧增
- 圣眷消耗是主要瓶颈——圣眷本身就稀缺且有戒心风险
- 操控2名考生的总消耗为上述数值×2
- 被操控的考生不限本家族成员，可操控任何考生（如拉拢其他家族）

**影响：**

| 隐藏分范围 | 作用 |
|-----------|------|
| 0.1~0.9（正常） | 不参与评分，仅同分时作为排序依据 |
| >1（被操控） | 直接加入评分公式计算，显著提升排名 |

**示例：**

- 省试正常：`综合分 = 学识 × 0.6 + 口才 × 0.4`，同分时比隐藏分
- 省试被操控：`综合分 = 学识 × 0.6 + 口才 × 0.4 + 隐藏分`（隐藏分>1时生效）
- 殿试正常：`综合分 = 学识 × 0.5 + 口才 × 0.3 + 声望排名分`，同分时比隐藏分
- 殿试被操控：`综合分 = 学识 × 0.5 + 口才 × 0.3 + 声望排名分 + 隐藏分`（隐藏分>1时生效）

#### 第二试：省试（礼部试）

**规则：**

- 全省（3路合并）考生统一排名
- 考察学识与口才两项属性
- 取综合分最高的前10名通过，进入殿试
- 未通过省试者保留解额2届，下次科举可免解试直接参加省试

**判定方式：**

```
正常：综合分 = 学识 × 0.6 + 口才 × 0.4
被操控：综合分 = 学识 × 0.6 + 口才 × 0.4 + 隐藏分（隐藏分>1时）
```

- 全路考生按综合分降序排列 → 取前10名通过
- 同分时比较隐藏分，隐藏分高者排前
- 隐藏分仍相同则随机决定

**解额保留规则：**

- 通过解试但未通过省试者，解额保留2届
- 保留期间内下次科举可免解试直接参加省试
- 超过2届未通过省试，解额作废，需重新参加解试

#### 第三试：殿试（皇帝亲试）

**规则：**

- 全省（3路合并）10名省试通过者统一排名
- 考察学识、口才与家族声望三项
- 排名前3为三甲，直接获官职
- 第4~10名获做官身份 + 50声望奖励，需通过空缺填补机制获取官职

**判定方式：**

```
正常：殿试综合分 = 学识 × 0.5 + 口才 × 0.3 + 声望排名分
被操控：殿试综合分 = 学识 × 0.5 + 口才 × 0.3 + 声望排名分 + 隐藏分（隐藏分>1时）
```

**声望排名分计算：**

- 同家族考生声望排名分相同
- 按家族声望在殿试考生中排序

| 家族声望排名 | 排名分 |
|-------------|--------|
| 第1名家族 | 100 |
| 第2名家族 | 80 |
| 第3名家族 | 60 |
| 第4名家族 | 40 |
| 第5~10名家族 | 20 |

- 10名殿试考生按综合分降序排列 → 第1~3名为三甲
- 同分时比较隐藏分，隐藏分高者排前
- 隐藏分仍相同则比较声望排名分，高分者排前

#### 殿试三甲与官职

| 等级 | 名称 | 人数 | 结果 |
|------|------|------|------|
| **一甲** | 进士及第 | 1人 | 直接获官职（从八品起） |
| **二甲** | 进士出身 | 1人 | 直接获官职（正九品起） |
| **三甲** | 同进士出身 | 1人 | 直接获官职（从九品起） |
| **未入三甲** | — | 7人 | 获做官身份 + 50声望，需通过空缺填补机制获取官职 |

#### 三甲官职分配机制

按优先级依次判定：

1. **有空缺**：对应品级有空缺官职则直接填补
2. **替换丙等者**：对应品级无空缺时，若有去年述职丙等者，直接替换，被替换者变为散官回家
3. **资源点替换**：前两者都不满足时，家族需投入资源点替换在任者，被替换者变为散官回家

#### NPC家族科举行为

- NPC家族中符合条件的成员由AI自动报名参加科举
- 报名费从NPC家族金钱中扣除
- NPC考生的能力值按其角色卡属性参与判定，与玩家同等规则

#### 科举全流程示例

```
[科举年8月] 报名解试（5贯/人，7月报名）→ 学识≥50者获解额（无名额上限）
    ↓
[次年2月] 报名省试（5贯/人，1月报名）→ 学识×0.6+口才×0.4排序 → 前10通过
    ↓
[次年3月] 报名殿试（5贯/人，2月报名）→ 学识×0.5+口才×0.3+声望排名分排序 → 三甲获官职
    ↓
其余7人获做官身份+50声望 → 等待空缺填补
```

### 地方官升迁：政绩槽机制

| 要素 | 设计 |
|------|------|
| **政绩槽** | 地方官职专属，与官职绑定而非角色；中枢官职无政绩槽，改为稳固值机制 |
| **填充速率** | 基础速率由政绩槽依据能力决定（见下方公式） |
| **速率加成** | 声望、党派支持、资源投入、圣眷均对速率有加成 |
| **事件影响** | 事件可直接增减政绩槽进度 |
| **年度述职** | 每年回京述职，评价政绩槽填充程度 |
| **升迁条件** | 任意连续两年甲等（槽填充≥100%）方可升迁 |
| **连升机制** | 连续两年甲等时，根据政绩槽超额程度决定连升阶数：两年均≥100%升1阶；两年均≥150%升2阶；两年均≥200%升3阶 |
| **升迁方向** | 连续两年甲等后可选择进入中枢或继续留任地方（品级更高） |
| **中枢后** | 进入中枢官职后无政绩槽，改为稳固值机制 |

### 年度述职升降阶处理流程

年度述职时，先处理降阶再处理升阶，具体流程如下：

1. **收集降阶池**：政绩槽不达标的官员，从当前官职上拿下，放入降阶池
2. **处理升阶**：按连升阶数从多到少排序处理（升3阶→升2阶→升1阶）
   - 有空缺则直接填补
   - 无空缺时，按同品阶官员去年政绩槽排序，末位者放入降阶池，腾出位置给升阶者
3. **处理降阶池**：降阶池中官员按原官阶从高到低排序，同阶按去年政绩槽排序，依次挑选比自己原官位低的空缺填补
4. **无位可补**：降阶池中未能填补任何空缺的官员，视为罢免

### 政绩槽填充速率公式

**月填充量：**

```
月填充量(%) = 基础填充 × 能力系数 × (1 + 加成合计)

基础填充 = 8.33%（即100% / 12月，能力满值时刚好一年填满）

能力系数 = 依据能力加权平均值 / 100

依据能力加权 = 各官职"政绩槽依据"对应能力的加权平均
  - 双依据官职：主依据×0.6 + 副依据×0.4
  - 三依据官职：主依据×0.4 + 次依据×0.3 + 副依据×0.3
```

**各官职政绩槽依据与权重：**

| 官职 | 依据1(×0.6/0.4) | 依据2(×0.4/0.3) | 依据3(×0.3) |
|------|------------------|------------------|-------------|
| 知县事 | 学识 | 声望(家族声望/100) | — |
| 主簿 | 商业 | 学识 | — |
| 县尉 | 军事 | 学识 | — |
| 知州事 | 学识 | 声望(家族声望/100) | — |
| 通判 | 学识 | 口才 | — |
| 司户参军 | 商业 | 学识 | — |
| 司理参军 | 军事 | 学识 | — |
| 转运使 | 商业 | 口才 | 学识 |
| 提点刑狱 | 军事 | 口才 | 学识 |
| 安抚使 | 军事 | 口才 | 声望(家族声望/100) |
| 宣抚使 | 军事 | 口才 | 声望(家族声望/100) |

- "声望"作为依据时取家族声望/100（如声望3000则代入30），避免声望数值量级碾压能力值

**加成合计：**

```
加成合计 = 党派支持加成 + 资源投入加成 + 圣眷加成 + 特性/头衔加成

党派支持加成 = 同党派地方官占该州/路比例 × 10%（上限20%）
资源投入加成 = 投入资源点 × 5%/点（上限15%，每回合最多投3点）
圣眷加成 = 投入圣眷 × 2%/点（上限10%，每回合最多投5点）
特性/头衔加成 = 经世"政务效率+10%"等对应特性/头衔效果累加
```

**填充示例：**

| 场景 | 能力系数 | 加成 | 月填充 | 年填充 |
|------|---------|------|--------|--------|
| 知县(学识60,声望1000→10) | (60×0.6+10×0.4)/100=0.40 | 党派10%+投入2点10% | 8.33%×0.40×1.20=4.0% | 48% |
| 知县(学识90,声望5000→50) | (90×0.6+50×0.4)/100=0.74 | 党派20%+投入3点15%+圣眷3点6% | 8.33%×0.74×1.41=8.7% | 104% |
| 转运使(商业80,口才70,学识60) | (80×0.4+70×0.3+60×0.3)/100=0.71 | 党派20%+投入3点15%+圣眷5点10% | 8.33%×0.71×1.45=8.6% | 103% |

- 中等能力(60分档)无加成时年填充约40%~48%，需加成才能甲等
- 高能力(90分档)+多项加成可稳定甲等
- 低能力官员需要大量资源投入才能勉强甲等，否则面临丙等/丁等风险

### 年度述职评级

| 等级 | 条件 | 效果 |
|------|------|------|
| **甲等** | 政绩槽 ≥ 100% | 升迁必要条件 |
| **乙等** | 80% ≤ 政绩槽 < 100% | 安全，但不计入升迁 |
| **丙等** | 60% ≤ 政绩槽 < 80% | 警告，连续两年丙等→罢免 |
| **丁等** | 政绩槽 < 60% | 直接罢免 |

### 关键规则

- **升迁**：任意连续两年甲等方可升迁
- **连升**：两年均≥150%可连升2阶，两年均≥200%可连升3阶
- **罢免触发**：连续两年丙等 或 有一年丁等
- **政绩槽重置**：每年述职后清零重新累积

### 政绩槽攻防规则

详见[argue_game_party.md](argue_game_party.md) → 8.2节

### 地方官任期

- 地方官无任期限制，只要不被罢免即可一直担任
- 升迁后自动离开原官职，原官职产生空缺

### 散官机制

- 散官：有品级但无官职的官员
- 保留品阶和俸禄
- 资历仍会缓慢降低（与罢免在家相同）
- 可通过空缺填补机制重新获取官职（需根据资历判定投入资源点）

### 女性官员孕期机制

由于此游戏中男女均可科举做官，女性官员在怀孕期间适用以下特殊规则：

**触发条件：** 女性官员怀孕判定成功后，从怀孕第6个月开始生效

**孕期时间线：**

```
怀孕判定成功 → 前6个月：正常担任官职，享受全部收益
              → 第7~10个月（共4个月=4回合）：进入孕期休假状态
              → 第10个月孩子出生：休假结束，恢复正常官员状态
```

**孕期休假状态规则：**

| 要素 | 规则 |
|------|------|
| **角色位置** | 从官职上回到家族中 |
| **官职归属** | 视为该角色依然在官职上，官职不产生空缺 |
| **稳固值/政绩槽** | 依然可以被攻击和防御 |
| **官职收益** | 不享受官职提供的任何收益（俸禄、圣眷产出、家族加成等） |
| **政绩槽填充** | 孕期休假期间政绩槽不增长 |
| **资历** | 孕期休假期间资历视同散官，会逐渐降低 |
| **稳固值被攻破** | 若女性官员在中枢官职上，休假期间稳固值被攻击降至0，同样会被贬官或罢免 |

**休假结束：**

- 孩子出生后，角色自动恢复官员状态，回到官职上，重新享受全部收益
- 无需通过空缺填补机制重新获取官职

### 地方官空缺填补

- 官职因死亡/罢免/升迁产生空缺后，进入空缺填补流程
- 未担任官职的角色需完成过科举方可获取空缺（入仕）
- 不同获取方式所需资源点不同：

| 获取方式 | 说明 | 资源点消耗 |
|----------|------|------------|
| **升迁** | 从低品级官职升任高品级空缺 | 中等 |
| **自降获取** | 主动降品级获取空缺官职 | 最低 |

#### 地方官填补规则：条件优先制

- 地方官空缺按以下优先级自动分配：

| 优先级 | 条件 | 说明 |
|--------|------|------|
| **1** | 升迁 | 在任官员升入空缺，优先级最高 |
| **2** | 自降获取 | 高品级散官主动降级填补 |
| **3** | 新入仕 | 通过科举获得做官身份者填补 |

- 同一优先级内多个竞争者时，比较资源点投入，投入最高者获得
- 同一优先级内资源点投入也相同时，比较资历，资历高者优先
- 空缺产生后1个月内完成填补

- 罢免/散官在家后重新入仕，需根据资历判定需投入的资源点

### 中枢官职空缺填补

- 中枢官职空缺采用**暗标制**：
- 各家族同时投入资源点+圣眷+声望，互相不可见对方投入
- 投入总值最高者获得该官职

**暗标计算公式：**

```
投入总值 = 资源点 × 10 + 圣眷 × 1 + 声望 × 0.5
```

- 仅限已完成科举且品级不低于该官职下一阶的角色参与竞标
- 空缺产生后1个月内完成竞标，竞标失败的资源点/圣眷/声望全额退还

### 资历系统

- 角色隐藏属性，不显示给玩家
- 数值范围0~100
- 用于判定重新入仕时需投入的资源点数量

**资历增长公式：**

```
担任官职时每月增长 = 品阶权重

品阶权重：
  正一品=1.5, 从一品=1.3, 正二品=1.1, 从二品=0.9
  正三品=0.7, 从三品=0.5, 正四品=0.4, 从四品=0.35, 正五品=0.3, 从五品=0.25
  从五品以下=0.2
```

**资历衰减公式：**

```
罢免/散官在家时每月衰减 = 1

资历最低为0，不会降为负值
```

**资历与重新入仕消耗：**

```
重新入仕所需资源点 = max(1, round((100 - 资历) / 10))

示例：
  资历90 → 需1资源点
  资历70 → 需3资源点
  资历50 → 需5资源点
  资历20 → 需8资源点
  资历0  → 需10资源点
```

- 升迁、调任、自降获取均需额外消耗上述资源点
- 散官自动填补空缺时也消耗资源点（从家族资源点中扣除）
- 科举三甲直接获官职不消耗资源点
- 资历越低，重新入仕所需投入的资源点越多

## 十一、系统九：任务系统

### 任务类型

| 类型 | 说明 |
|------|------|
| **主线任务** | 与剧情绑定，推动复仇主线，有期限，失败则游戏结束 |
| **随机任务** | 每阶段按模板自动生成，有期限，失败仅丢失奖励 |
| **党派任务** | 加入党派后按模板生成，有期限，失败丢失奖励+少量忠诚度 |

### 主线任务

主线任务与剧情文档紧密绑定，每个剧情阶段对应一组主线任务：

| 阶段 | 主线任务示例 | 期限 | 失败后果 |
|------|------------|------|---------|
| 第3期·科举 | 通过科举 | 3年内 | 游戏结束 |
| 第4期·地方 | 升迁至从三品（宣抚使） | 10年内 | 游戏结束 |
| 第5期·中枢 | 查明灭门真相 | 8年内 | 游戏结束（遇刺） |
| 第6期·复仇 | 击败主谋 | 8年内 | 游戏结束（贬官/流放） |

> 主线任务的详细内容、触发条件、完成条件详见剧情文档 [argue_game_diag.md](argue_game_diag.md)

### 随机任务

每阶段有对应的任务模板池，系统定期从中抽取任务。

**生成规则：**

| 阶段 | 生成频率 | 同时进行上限 |
|------|---------|------------|
| 地方官阶段 | 每6个月生成1个 | 2个 |
| 中枢阶段 | 每4个月生成1个 | 3个 |
| 党派领袖阶段 | 每3个月生成1个 | 3个 |

**任务模板：**

| 阶段 | 模板名 | 目标 | 期限 | 奖励 |
|------|--------|------|------|------|
| 地方官 | 治水安民 | 政绩槽累计达200% | 2年 | 金钱+80贯, 声望+200 |
| 地方官 | 赈灾募捐 | 金钱累计支出≥200贯 | 1年 | 声望+300, 地方关系+10 |
| 地方官 | 肃清盗匪 | 军事≥50的角色完成3次任务 | 2年 | 声望+150, 金钱+50贯 |
| 地方官 | 兴学重教 | 家族私塾/书院累计放置≥12月 | 2年 | 学识成长+10%(永久), 声望+100 |
| 地方官 | 商路开拓 | 城市建筑金钱累计产出≥500贯 | 3年 | 金钱+200贯, 商业+5 |
| 中枢 | 朝堂建言 | 攻讦/弹劾成功≥3次 | 2年 | 圣眷+20, 党派贡献+20 |
| 中枢 | 联络同僚 | 与3个同党派家族关系≥60 | 2年 | 声望+300, 党派贡献+15 |
| 中枢 | 巩固根基 | 己方稳固值累计恢复至80+共3次 | 2年 | 圣眷+15, 稳固值+10 |
| 中枢 | 纳贤入幕 | 联姻成功≥2次 | 3年 | 角色卡×1, 声望+200 |
| 中枢 | 暗中布局 | 情报卡累计领取≥5张 | 1年 | 情报卡升级免费1次, 党派贡献+10 |
| 党派领袖 | 党同伐异 | 累计击倒/贬官敌方官员≥2人 | 3年 | 党派声望+200, 圣眷+30 |
| 党派领袖 | 招降纳叛 | 策反1个敌方家族 | 3年 | 该家族加入己方党派, 党派声望+300 |
| 党派领袖 | 执掌中枢 | 己方党派占据中枢官职≥10个 | 2年 | 党派声望+500, 全族圣眷+20 |
| 党派领袖 | 家族壮大 | 家族声望≥7000（顶级家族） | 3年 | 家族人数上限+5, 建筑上限+2 |

- 随机任务从对应阶段模板池中等概率抽取
- 已完成的模板不会重复抽取
- 所有模板完成后不再生成随机任务

### 党派任务

加入党派后额外生成的任务，与党争相关。

**生成规则：** 每6个月生成1个，同时进行上限2个

**任务模板：**

| 模板名 | 目标 | 期限 | 奖励 | 失败惩罚 |
|--------|------|------|------|---------|
| 攻击指定目标 | 将指定敌方官员稳固值降至X以下 | 1年 | 党派贡献+25 | 忠诚度-5 |
| 防御指定目标 | 己方指定官员稳固值不低于X持续6月 | 1年 | 党派贡献+20 | 忠诚度-3 |
| 情报收集 | 领取情报卡≥3张 | 6月 | 党派贡献+10 | 无 |
| 拉拢策反 | 将指定家族忠诚度降至30以下 | 2年 | 党派贡献+30 | 忠诚度-5 |
| 舆论攻势 | 发布2次舆论buff | 1年 | 党派贡献+15 | 无 |
| 资源上缴 | 向党派私库存入≥100贯 | 6月 | 党派贡献+10, 忠诚度+5 | 忠诚度-3 |

- 党派任务目标根据当前局势动态选择（如指定"稳固值最低的敌方官员"）
- 失败惩罚仅影响党派忠诚度，不导致游戏结束

### 任务奖励（混合）

| 奖励类型 | 示例 |
|----------|------|
| **资源奖励** | 金钱、声望、圣眷、党派贡献等 |
| **解锁奖励** | 新剧情、新功能、新角色卡等（仅主线任务） |
| **永久加成** | 成长加成、家族上限提升等（稀有奖励） |

### 失败机制

- **所有任务都有期限**
- 主线任务失败 → 游戏结束
- 随机任务失败 → 仅丢失奖励，无额外惩罚
- 党派任务失败 → 丢失奖励 + 少量忠诚度惩罚

## 十二、系统间关联总览

```
时间流逝系统（骨架）
    ↓ 每回合推进 + 特殊节点(述职/大朝会/科举)
资源系统 ← 家族系统（建筑产出资源点 + 俸禄）
    ↓ 资源分配        ↓ 角色卡放入建筑      ↓ 家族行动(联姻/结交)
人物/角色系统 → 官场系统（能力影响政绩槽/稳固值）
    ↓ 传承/获取/移除         ↓ 地方官政绩槽 → 中枢稳固值
党争系统 ←——————→ 官场系统（攻防稳固值/党派升迁）
    ↓ 党派机构/行动/忠诚
事件系统（多维度影响所有系统，分段事件池，稀有度）
    ↓
剧情/对话系统（混合解锁推进，多结局）
    ↓
任务系统（驱动目标与奖励，期限机制）
    ↓
失败重开（继承buff重开）
```

## 十四、失败重开

- 游戏失败后可重新开局
- 继承buff：**声望获取×1.2 + 金钱获取×1.2**
- 重新开局跳过新手教学阶段（第1期），直接从第2期灭门之灾开始
- buff可叠加：每次失败重开额外+0.2倍率（第2次失败重开为×1.4，第3次为×1.6，以此类推）
